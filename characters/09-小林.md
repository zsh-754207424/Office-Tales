# 小林 - 迷失的灵魂原型

## 基础信息

| 字段 | 内容 |
|------|------|
| **姓名** | 林晓东（小林） |
| **职位** | 实习生（你之后入职） |
| **年龄** | 22岁 |
| **性格** | 老实、勤奋、迷茫、缺乏主见 |
| **原型** | 迷失的灵魂（Lost Soul） |
| **出场** | 第1章（同期入职）/ 第5章（结局对照） |

## 外表印象

- 175cm，瘦高，戴眼镜
- 学生气很重
- 说话有点怯懦
- 总是拿个小本本记东西
- 看起来很努力但方向不对

## 人物关系

```
与主角：后辈/可能的被指导者（初始好感70）
    │
与王浩：被笼络（王浩在建立自己的小团队）
    │
与张姐：可能被你托付（如果你信任张姐）
```

## 核心动机

### 显性动机
- 留下来（转正）
- 学习工作
- 不出错

### 隐性动机
- 他不知道自己真正想要什么
- 他只是"不想让人失望"
- 他在寻找方向

## 迷失的灵魂特征

```
【迷失的灵魂定义】
- 代表主角可能变成的样子
- 从未达成目标的人
- 主角看到他会想"如果不努力，我也会这样"

【小林的功能】
- 是你的对照（另一个"职场小白"）
- 是你的镜子（你曾经也是这样）
- 是你的责任（你成功后要带他吗？）
```

## 关键剧情节点

### 第1章：同期入职
- 你和小林几乎同时入职
- 他看起来比你更紧张
- 你可以选择：帮他 / 不管他

### 第3章：开始分化
- 你开始成长
- 他还在原地
- 你可以选择：带他 / 让他自己摸索

### 第5章：结局对照
- 你的结局揭示后
- 他的结局也会展示
- 对比：你选择了什么，他选择了什么

## 对话风格

### 求助时
```
"前辈，这个怎么弄？"
"对不起，我又搞砸了"
"我是不是不适合这里？"
```

### 迷茫时
```
"我不知道该做什么"
"大家都好厉害，我跟不上"
"前辈，你当初是怎么过来的？"
```

### 感谢时
```
"谢谢前辈"
"你帮了我很多"
"我会努力的"
```

## 小林的3种结局

### 结局A：你成功了，你带他
```
你的结局：A1类（晋升）
小林的结局：在你的团队里成长

对话：
"前辈，谢谢你当时带我"
"我现在终于懂了"
（小林成为你的下属，延续你的路）
```

### 结局B：你失败了，你们一起失败
```
你的结局：C类（离职）
小林的结局：也离职了

对话：
"前辈，我们接下来怎么办？"
"也许我们不该来这家公司"
（你们变成"迷失的灵魂"，成为彼此的镜子）
```

### 结局C：你成功了，他选择另一条路
```
你的结局：A1类（晋升）
小林的结局：主动离开，去做其他事

对话：
"前辈，我想清楚了，这不是我想要的"
"我打算去考研/考公/创业"
（他找到了自己的路，和你不同）
```

## 你的选择如何影响他

### 第2章
```
你帮他：好感+20，他开始信任你
你不管他：他自己摸索，可能出错
```

### 第3章
```
你带他：他会成为你的支持者
你让他自己来：他可能被王浩笼络
你告诉他真相：他对职场有了清醒认识
```

### 第4章
```
裁员时你帮他说话：可能影响裁员结果
你不管他：他可能成为被裁的那个
```

## 隐藏属性

```yaml
背景：普通一本，家境普通
性格：讨好型人格，不敢拒绝
问题：努力但方向不对
真实想法："我真的适合这里吗？"
```

## 他代表的可能性

```
小林不是坏人，不是对手
他是另一个可能性的你

如果你选择：
- 努力但方向不对 → 变成他
- 帮助他 → 你有责任感
- 不管他 → 你更专注自己

他是你的镜子：
- 看到他的迷茫，你会想起自己
- 看到他的成长，你会欣慰
- 看到他的失败，你会警惕
```

## 设计注意事项

- 他不是主要角色，但有重要功能
- 他的存在让游戏不只是"你"的故事
- 他的结局是你选择的对照
- 他代表"普通人的职场路"
- 他的存在让游戏更有温度
- 玩家可以选择带他，也可以选择不管他
- 他是设计中最"真实"的角色
