## 内心生活

### 核心身份

```
【天赋】
- 社交：认识所有人，上到董事长，下到保安
- 权谋：知道谁需要谁，谁怕谁
- 自控：永远不表现真实情绪
- 信息：永远比别人知道得多

【缺陷】
- 不在乎能力，只在乎关系
- 实用主义：什么都能交易
- 操纵：让别人按他想要的方式做事
- 嫉妒：看不起"靠能力上位的人"

【信仰】
- 人际观："所有关系都是交易"
- 成功观："有人脉就能成功"
- 道德观："规则是给普通人定的"
```

### 信念系统

```
【道德底线】
- 不违法（表面上）
- 不留下证据
- 关系是唯一的货币
- 忠诚是可以买到的

【定式/偏见】

【偏见来源】

- 他发誓要赢回来
```

【最大野心】

- 让那些"看不起他"的人后悔
- 证明：关系比能力更重要

【最深的恐惧】

## 决定性特征

```
【操纵者】
一切都可交易，一切都有价码——这是他的信条

【关系玩家】
他不是靠能力上位的，但他不觉得这有问题
他相信：这就是职场的规则
```

### 经历塑造

```
【催化剂事件】

事件1：第一次被排挤
- 刚入职时，他做了很多项目
- 但年底晋升的是另一个同事
- 那个同事说："你能力不错，但没有背景"
- 他第一次明白：在这个世界，能力不是最重要的

事件2：娶了"有背景"的老婆
- 相亲认识的，条件很好
- 婚后他升得很快
- 他从来不觉得这是"吃软饭"
- 他觉得这是"聪明的选择"

事件3：十年前的总监之争
- 他和李总监同时竞争总监位置
- 他的方案比李总监好
- 但最后是李总监赢了
- 他知道是谁在背后支持李总监
- 但他没有证据
- 从此，他和李总监成了死对头

事件4：陈叔的事
- 陈叔是他的"棋子"之一
- 当年陈叔站他这边
- 最后他赢了，但陈叔被他抛弃了
- 他不后悔：这就是职场的规则

【深度认识】

他认识到：
- 在这个世界上，能力是次要的，关系才是主要的
- 所有人都可以被收买，只是价格不同
- 规则是给普通人定的，有关系的人不需要遵守
- 忠诚是可以买到的，但也可以随时抛弃

他决定：
- 继续走这条路
- 培养自己的人
- 等机会超过李总监

【承担个人责任】

他选择：
- 不承认自己"吃软饭"
- 不承认自己"靠关系"
- 把一切都归结为"聪明"和"选择"
- 他不觉得自己做错了什么——这只是"生存法则"
```
