## 出场顺序

### 第1章：入职适应
- 张姐（主动接触）
- 王浩（同期竞争）
- 李总监（首次汇报）
- 小林（新人对照）

### 第2章：首个任务
- 陈叔（可解锁）
- 小刘HR（入职手续）
- 王浩（抢功事件）

### 第3章：团队协作
- 秦总（远观）
- Linda（跨部门）
- 张姐（站队选择）
- 王浩（持续对抗）

### 第4章：利益冲突
- 陈叔（核心情报）
- 小刘HR（裁员预警）
- 王副总（拉拢/威胁）
- 王浩（最终对决）

### 第5章：结局
- 李总监（最终评判）
- 秦总（终极对话）
- 小林（结局对照）
- 全角色（关系总览）
