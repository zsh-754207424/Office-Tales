# 王副总 - 影子原型

## 基础信息

| 字段 | 内容 |
|------|------|
| **姓名** | 王建国 |
| **职位** | 副总经理 |
| **年龄** | 45岁 |
| **性格** | 圆滑、城府深、手段高明、笑面虎 |
| **原型** | 影子（Shadow） |
| **出场** | 第4章（拉拢/威胁） |

## 外表印象

- 175cm，微胖，永远笑容满面
- 穿着比李总监随意，但品味好
- 说话和气，像个好人
- 但笑容背后是计算
- 被称为"最和善的领导"

## 人物关系

```
与主角：想拉拢/想打压（取决于你的选择）
    │
与李总监：派系对立（明争暗斗10年）
    │
与王浩：暗中指挥（王浩是他的人）
    │
与陈叔：曾经的对手（他赢了那场）
    │
与秦总：表面服从，暗中博弈
```

## 核心动机

### 显性动机
- 扩大自己的势力
- 压制李总监
- 为下一次晋升铺路

### 隐性动机
- 他觉得自己才是对的
- 他用"关系"而非"能力"评价人
- 他想证明自己的路才是正道

## 影子特征

```
【影子的定义】
- 暴露主角的性格缺陷
- 代表主角不想成为的样子
- 力量与主角相当或更强

【王副总的影子功能】
- 代表：不择手段的野心
- 代表：关系的价值 > 能力
- 代表：踩着别人往上爬

【你看到他的反应】
"我不想变成那样"
"但为了生存，我是不是也会……"
```

## 关键剧情节点

### 第4章：拉拢（核心场景）
- 他会私下找你谈话
- "小年轻，跟着李总监没前途的"
- "王浩已经是我的人了，你要不要也考虑考虑？"
- "我知道你和王浩有点过节，我们可以帮你"

### 第5章：结局对决
- 你的选择决定他是敌是友
- 拒绝他：成为他的眼中钉
- 接受他：你的价值观会改变

## 对话风格

### 表面和善
```
"小王啊，年轻有为！"
"有什么事尽管来找我"
"我看好你"
```

### 拉拢时
```
"我听说你和李总监有些分歧？"
"年轻人有想法是好事"
"但选对路更重要"
"王浩已经想通了，你呢？"
```

### 威胁时（如果你拒绝）
```
"年轻人，有个性是好事"
"但个性不能当饭吃"
"希望你想清楚"
（然后意味深长地笑笑）
```

## 他的行事风格

```
【他的手段】
- 从不正面冲突
- 用"关系"而非"能力"评价人
- 让你主动选择站队
- 给你甜头，然后要你回报
- 记仇，但不会当场发作

【他的底线】
- 不能动他的人
- 不能挑战他的权威
- 不能让他丢面子
```

## 隐藏属性

```yaml
背景：靠关系上位，但也有能力
团队：氛围好，大家"一团和气"
秘密：知道李总监的"把柄"但不会用
真实想法："能力重要还是关系重要？时间会证明我是对的"
```

## 拉拢选项及后果

### 接受拉拢
```
→ 获得王副总的支持
→ 王浩成为你的"同事"（关系缓和）
→ 李总监不再信任你
→ 价值观改变：开始相信"关系比能力重要"

后果：
- 短期：晋升更快
- 长期：可能变成下一个"王副总"
```

### 拒绝拉拢
```
→ 王副总成为敌人
→ 王浩对你更敌视
→ 李总监更信任你
→ 坚守自己的价值观

后果：
- 短期：被针对
- 长期：可能离开，但也保留了自我
```

### 不表态
```
→ 王副总觉得你"还在考虑"
→ 保持中立
→ 但中立在这里很难

后果：
- 两边都不信任你
- 成为"墙头草"
```

## 设计注意事项

- 他不是纯粹的坏人，他有自己的逻辑
- 他的"关系论"在这个社会确实有效
- 他让玩家思考"手段和目的哪个重要"
- 他的存在让职场不是简单的"能力vs能力"
- 他是主角价值观的试金石
- 选他还是选李总监，代表了两种职场哲学
