## 内心生活

### 核心身份

```
【天赋】
- 听话：让做什么就做什么
- 吃苦：愿意加班，愿意做杂活
- 真诚：还不会演戏

【缺陷】
- 没主见：别人说什么就是什么
- 自我怀疑：经常问自己是不是不行
- 逃避：用"迷茫"来逃避选择

【信仰】
- 成功观："成功的人一定很辛苦"
- 工作观："先就业再择业"
- 家庭观："父母为我付出太多了"

【身份认同】
- 还没有找到自己是谁
- 活在父母的期待里
- 实习生=还没长大的孩子
```

【定式/偏见】

【偏见来源】

## 成长弧线

### 入职前

```
- 普通一本毕业，专业被调剂
- 考研失败，理由是"没发挥好"
- 找工作时发现：好工作需要能力，普通工作不需要他
- 终于找到这份实习

【这个阶段塑造了他】
- "我是普通人"的心理定势
- 对未来的迷茫
- 对父母期望的愧疚
```

### 入职初期

```
- 被主角帮助（如果玩家选择了A）
- 第一次感受到有人愿意帮他
- 但他不知道该怎么回报

【这个阶段塑造了他】
- 主角成为了他的参照
- 他开始思考："我想变成什么样的人？"
```

### 发展期

```
- 主角被王浩针对
- 他看到了职场的黑暗面
- 他面临选择：站哪边？

【这个阶段塑造了他】
- 他可能选择站王浩（为了安全）
- 他可能选择站主角（为了感恩）
- 这个选择定义了他是谁
```

### 现在

```
- 实习期快结束了
- 要么转正，要么离开
- 他必须做出选择

【现在的他】
- 比入职时更迷茫
- 但也比入职时更清醒
- 他开始问自己："我想成为什么样的人？"
```

## 经历塑造

### 催化剂事件

```
【推动改变的催化剂】

事件1：主角的帮助
- 如果主角帮了他
- 第一次有人把他当回事
- 他感受到了温暖

事件2：王浩的接近
- 王浩给他好处
- 他不知道该不该接受
- 他开始思考"对和错的标准"

事件3：裁员危机
- 他发现自己也在名单上
- 第一次感受到生存压力
- "原来这就是职场"
```

### 深度认识

```
【认识他人阶段】

他认识到的：
- 主角是真心帮他
- 王浩只是在利用他
- 张姐有她的难处

【认识自我阶段】

他认识到：
- 他一直在逃避
- 他想依赖别人做决定
- 他必须自己承担责任

【承担个人责任】

他决定：
- 站主角那边（因为主角帮过他）
- 或者站王浩那边（因为现实需要）
- 或者自己离开（因为不想参与）
```

### 行动

```
【基于认识的行动】

行动1：如果站主角
- 他会告诉主角他知道的信息
- 他会在关键时刻为他说一句话
- 这是他第一次为自己选择

行动2：如果站王浩
- 他会默默接受王浩的安排
- 他会告诉自己"这是现实"
- 但内心会有愧疚

行动3：如果离开
- 他会选择主动辞职
- 他会去找真正适合自己的路
- 这是他第一次为自己做决定
```

## 与主角的关系弧线

### 第1章：初遇

```
【表面】
"前辈好，我是小林……"

【实际】
在观察：
- 主角和他一样是新人
- 但主角比他自信多了

【深层】
"如果我能像主角一样就好了"
```

### 第2章：被帮助

```
【表面】
"谢谢前辈……"

【实际】
- 第一次有人主动帮他
- 他感受到了温暖
- 他不知道怎么回报

【深层】
"也许……我不是一个人"
```

### 第3章：选择

```
【表面】
"前辈，王浩在接近我……"

【实际】
- 他在问主角该怎么办
- 他需要有人替他做决定
- 但主角可能不帮他

【深层】
"我想站你这边，但我怕"
```

### 第4章：命运

```
【表面】
"前辈，我……我要离开了"

【实际】
- 他的选择决定了他的命运
- 他可能在主角的帮助下留下
- 也可能因为站错队而离开

【深层】
"不管结果如何，这是我选的"
```

### 可能的结局

```
【留下来了】
- 继续迷茫，但也继续成长
- 有了主角这个参照
- 开始慢慢找到自己的路

【离开了】
- 换一家公司重新开始
- 或者去考研/考公
- 至少找到了一个方向
```
