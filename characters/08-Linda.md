# Linda - 逗乐者原型

## 基础信息

| 字段 | 内容 |
|------|------|
| **姓名** | 林小芳（Linda） |
| **职位** | 财务专员 |
| **年龄** | 27岁 |
| **性格** | 活泼、话多、爱八卦、嘴碎但心不坏 |
| **原型** | 逗乐者（Jester） |
| **出场** | 第3章（跨部门接触） |

## 外表印象

- 160cm，圆脸，爱笑
- 穿着亮丽，喜欢彩色
- 说话快，像机关枪
- 笑声很魔性
- 永远在嚼口香糖或喝奶茶

## 人物关系

```
与主角：茶水间偶遇（初始好感55）
    │
与各部门：消息灵通（情报站）
    │
与王浩：聊得来（但不代表支持他）
    │
与张姐：认识但不熟
    │
与主角：可能成为朋友，但要注意风险
```

## 核心动机

### 显性动机
- 做好财务工作
- 聊八卦、放松
- 和大家保持好关系

### 隐性动机
- 通过消息灵通获得安全感
- 享受"被人需要"的感觉
- 其实她也有职场焦虑

## 逗乐者特征

```
【逗乐者能力】
- 喜剧调剂（紧张氛围的润滑剂）
- 情报中心（但真假参半）
- 侧面观察（旁观者清）

【逗乐者风险】
- 她说的话可能不准确
- 你说的话可能传出去
- 她可能无意中出卖你
```

## 关键剧情节点

### 第3章：首次接触
- 茶水间偶遇
- 她主动和你聊天
- 这是建立关系的机会

### 第3章：跨部门合作
- 财务需要配合的项目
- 和她打交道的机会
- 可以获取财务视角的信息

### 第4章：情报来源
- 她可能无意中透露一些消息
- 但要会筛选
- 假消息比没消息更危险

## 对话风格

### 日常八卦
```
"你知道吗！王浩今天又被领导骂了！哈哈哈哈"
"李总监今天心情不好，大家绕着走"
"听说公司要发年终奖了！……开玩笑的"
```

### 提供信息时
```
"我听说……可能不准啊"
"这个消息你别告诉别人"
"我就随口说说"
```

### 好奇打探时
```
"你和王浩是不是有什么过节？"
"李总监对你怎么样？"
"你有没有听说……"
```

## 她的价值

### 1. 了解办公室气氛
```
"今天气氛不对，大家小心点"
"领导们开会很久了，可能有事"
```

### 2. 非正式信息
```
"我看到报销单了，最近项目费涨了"
"财务说这个季度回款不太好"
```

### 3. 侧面验证
```
"你说的那个事，我也听说了"
"但好像有点不一样……"
```

## 她的风险

### 1. 信息可能不准确
```
她说的80%是真的
但20%是夸大或误传
你要有判断能力
```

### 2. 你说的话可能传出去
```
今天和她说的
明天可能全公司都知道
保持友好，但不要交心
```

### 3. 她可能无意中出卖你
```
她不是故意的
但"我就随口问问"可能变成公告
```

## 与Linda的关系策略

### 策略A：友好交往
```
好处：
- 信息获取 +5
- 了解办公室气氛
- 休息时间的调剂

风险：
- 你说的话可能传出去
- 被人利用（有人会通过她打探你）

适合：信息筛选能力强的人
```

### 策略B：保持距离
```
好处：
- 风险 -20
- 不会被她无意出卖

风险：
- 信息获取 -10
- 可能显得不合群

适合：谨慎型玩家
```

### 策略C：利用她
```
好处：
- 获取特定信息
- 通过她传递消息

风险：
- 如果被发现，可能反噬
- 她可能变成敌人

适合：有心机型玩家
```

## 隐藏属性

```yaml
真实性格：开朗外表下也有焦虑
信息来源：财务系统 + 茶水间
心理状态：需要存在感，喜欢被需要
真实想法："我就随便说说，你别当真啊"
```

## 设计注意事项

- 她是紧张氛围的调剂
- 她让游戏不只是严肃的职场
- 她的信息有价值但要筛选
- 她可能无意中推动剧情
- 她的存在让办公室更"真实"
- 玩家可以选择和她交朋友，也可以保持距离
