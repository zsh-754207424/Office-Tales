# 李总监 - 指导者原型

## 基础信息

| 字段 | 内容 |
|------|------|
| **姓名** | 李建华 |
| **职位** | 部门总监 |
| **年龄** | 42岁 |
| **性格** | 结果导向、雷厉风行、外冷内热 |
| **原型** | 指导者（Mentor） |
| **出场** | 第1章（入职面试） |

## 外表印象

- 身高178cm，头发短促略显花白
- 永远西装笔挺，领带颜色不超过3种
- 说话简洁有力，从不废话
- 表情严肃，但偶尔会在不经意间流露一丝笑意
- 办公桌上永远整齐，只有电脑和一叠文件

## 人物关系

```
与主角：直属领导（初始信任50）
    │
与秦总：被提拔/被观察（秦总想看看他培养的人如何）
    │
与陈叔：前下属/旧识（知道陈叔为什么被边缘化）
    │
与王副总：竞争对手（派系对立）
    │
与张姐：信任的下属（核心班底）
    │
与王浩：观察中（王副总安插的人？）
```

## 核心动机

### 显性动机
- 带领部门完成业绩
- 培养能打硬仗的团队
- 在公司站稳脚跟

### 隐性动机
- 证明自己不比王副总差
- 想找一个"继承人"
- 不想再经历一次"当年没人带我"的孤独

## 行为模式

### 喜欢
- 主动汇报、不找借口
- 扛得住压力、敢于担责
- 结果说话，不听过程诉苦
- 加班但不抱怨（态度比时间重要）

### 厌恶
- 越级汇报（红线）
- 背后议论同事
- 推卸责任
- 找借口而不是找方案

## 关键剧情节点

### 第1章：入职面试
- 他决定招你入伙
- 面试问题："你为什么离开上一份工作？"
- 隐藏信息：他看到了曾经的自己

### 第2章：第一个任务
- 交给你一个中等难度的任务
- 观察你的工作方式
- 关键时刻：任务完成度决定初始信任值

### 第3章：季度评估
- 当众表扬或批评
- 给你一个展示机会
- 关键时刻：功劳分配方式

### 第4章：裁员危机
- 他掌握你命运的决定权
- 可能保护你，也可能放弃你
- 关键时刻：你的选择影响他的判断

### 第5章：最终抉择
- 提名晋升 / 维持现状 / 让你离开
- 取决于全程积累的信任值

## 对话风格

### 日常对话
```
"说重点。"
"我不想听理由，我想看结果。"
"有问题吗？有问题现在说。"
```

### 关键时刻
```
"我年轻的时候，也像你这样……"（然后打住）
"你知道为什么我选你吗？"
"别让我后悔。"
```

### 负面反馈
```
"你觉得这样合适吗？"
"我给你的机会，不是让你浪费的。"
"你可以走了。"
```

## 信任值系统

| 信任值 | 态度 | 对话变化 |
|--------|------|----------|
| 0-20 | 彻底放弃 | "这件事你不用跟了" |
| 21-40 | 失望观望 | "我看看你还能做什么" |
| 41-60 | 正常上下级 | 公事公办 |
| 61-80 | 重点培养 | "有个机会，你想不想试试" |
| 81-100 | 心腹嫡系 | "这件事我只跟你说" |

## 信任值变化规则

```
【增加信任】
+15 任务出色完成
+20 关键时刻站出来
+15 主动认错不甩锅
+10 给他有价值的情报
+10 维护团队利益

【降低信任】
-20 越级汇报
-30 推卸责任
-20 当众让他难堪
-25 背后议论他
-15 连续两次任务失败
```

## 隐藏属性

```yaml
健康状态：轻微高血压（压力大时会头疼）
婚姻状态：离异，前妻带孩子在国外
睡眠质量：差，靠安眠药
真实想法："我还能在这个位置待多久？"
```

## 结局走向

| 信任值 | 结局位置 |
|--------|----------|
| ≥80 | A1类：成为他的嫡系，获得晋升 |
| 50-79 | B1类：稳定发展，但天花板明显 |
| <50 | C2类：被边缘化或离职 |

## 设计注意事项

- 他的"冷"是保护色，内心其实在意每个下属
- 他不会主动表达关心，但会默默帮你
- 他对"态度"的要求 > 对"能力"的要求
- 他的故事暗示：他曾经也是被放弃的那个
