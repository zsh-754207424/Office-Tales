# 张姐 - 挚友原型

## 基础信息

| 字段 | 内容 |
|------|------|
| **姓名** | 张雨晴 |
| **职位** | 高级专员 |
| **年龄** | 33岁 |
| **性格** | 热心、聪明、边界感强、现实主义 |
| **原型** | 挚友（Friend） |
| **出场** | 第1章（午餐邀请） |

## 外表印象

- 165cm，干练短发，淡妆
- 温和的笑容，做事有条不紊
- 穿着得体但不张扬
- 办公桌整洁，效率高
- 说话温柔但有分量

## 人物关系

```
与主角：老员工，愿意带新人（初始好感60）
    │
与李总监：信任的下属（核心班底成员）
    │
与王浩：表面友好（试探他能不能被拉拢）
    │
与陈叔：敬重但不靠近（知道他的故事但不主动接触）
    │
与小林：可能会带的新人（观察中）
```

## 核心动机

### 显性动机
- 保住自己的位置
- 工作顺利，不惹麻烦
- 维护自己的小圈子

### 隐性动机
- 她曾经也是新人，被人帮过
- 她想看看这个新人"值不值得帮"
- 她有自己的生存策略，不会无条件付出

## 挚友特征

```
【挚友能力】
- 一直在场（稳定存在）
- 真心关爱（但有条件）
- 提供情感支持（但也可能给糟糕建议）
- 告诉你真实的声音（可能不好听）
```

## 关键剧情节点

### 第1章：午餐邀请（关键选择）
- 她主动邀请你吃午餐
- 试探你的性格和态度
- 这是建立关系的第一步

### 第2章：告诉你潜规则
- 告诉你领导的雷区
- 告诉你哪些事能做哪些不能
- 告诉你办公室的政治格局

### 第3章：站队时刻
- 你必须选择立场
- 她会观察你选谁
- 关键时刻：你们的关系可能改变

### 第4章：裁员危机
- 她会给你暗示和帮助
- 但她不会为了你得罪领导
- 她会看你"值不值得帮"

### 第5章：结局影响
- 高好感：成为长期盟友
- 中好感：保持点头之交
- 低好感：形同陌路

## 对话风格

### 日常友好
```
"第一天怎么样？还适应吗？"
"中午一起吃饭？我带你去个好地方"
"有什么不懂的可以问我"
```

### 提点时
```
"有些话我只说一遍"
"你要看清楚谁是你的敌人，谁可以利用"
"记住，在这里，不说话不代表没态度"
```

### 警告时
```
"你今天说的那句话，很危险"
"王浩那边，你小心点"
"领导今天心情不好，你别撞枪口"
```

### 关键时刻
```
"我不会替你扛事，但我可以告诉你路怎么走"
"你想清楚，站队了就回不了头"
"你是个聪明人，应该知道怎么选"
```

## 好感度系统

| 好感度 | 态度 | 能获得的支持 |
|--------|------|--------------|
| 0-40 | 疏远 | 仅公事公办 |
| 41-60 | 友善 | 偶尔提点 |
| 61-80 | 信任 | 主动分享情报 |
| 81-100 | 盟友 | 关键时刻帮助 |

## 好感度变化规则

```
【增加好感】
+15 接受午餐邀请
+10 主动请教问题
+15 帮她一个小忙
+20 站队时选择支持她
+15 在关键时刻没有背叛她
+10 你的成功让她觉得有面子

【降低好感】
-10 拒绝午餐邀请
-15 抢她风头
-20 为了讨好领导而疏远她
-20 在背后说她坏话
-30 让她为你背锅
```

## 她能给的情报

### 1. 领导雷区
```
"李总监最讨厌两种人：
一是找借口的，二是越级汇报的"
```

### 2. 同事性格
```
"王浩？表面功夫很厉害
你小心点，他不是善茬"
```

### 3. 派系格局
```
"这里分两派，李总监一派，王副总一派
你别想着两边讨好，选一边站"
```

### 4. 潜规则
```
"周报要写，周一早上9点前发
领导看不到你的工作，等于你没工作"
```

## 隐藏属性

```yaml
背景：3年前入职，也曾差点被裁
经历：李总监保了她，她记这份情
现状：站稳脚跟，有自己的小圈子
真实想法："我能帮你的就这么多，剩下的靠你自己"
```

## 挚友的可能冲突

根据原型设计，挚友可能产生以下冲突：

- 给出糟糕的建议（她可能判断失误）
- 不愿主角改变（她习惯了现状）
- 嫉妒主角的成就（如果主角成长太快）
- 主角与其他角色关系好时吃醋（她不是唯一的朋友）

**但在她的设计中，要避免这些问题：**
- 她的建议是"真实的声音"，不好听但有用
- 她支持主角改变，但提醒风险
- 她为你的成功高兴，但也会提醒你不要太张扬

## 结局走向

| 好感度 | 结局位置 |
|--------|----------|
| ≥80 | 长期盟友，你成功时她会为你高兴 |
| 50-79 | 点头之交，保持友好关系 |
| <50 | 形同陌路，你失败时她不会伸出援手 |

## 设计注意事项

- 她不是无条件帮你的，要看你"值不值得"
- 她的话不好听，但都是真话
- 她有自己立场和利益，不会为你得罪领导
- 她的帮助是"授人以渔"，不是"授人以鱼"
- 她是你在公司里最真实的声音
- 如果你背叛她，她会记住；如果她帮助你，你也要回报
