# 陈叔 - 智者原型

## 基础信息

| 字段 | 内容 |
|------|------|
| **姓名** | 陈德明 |
| **职位** | 资深经理（已边缘化） |
| **年龄** | 52岁 |
| **性格** | 看透一切、淡泊名利、偶尔提点、孤独 |
| **原型** | 智者（Sage） |
| **出场** | 第2章（可主动拜访） |

## 外表印象

- 头发花白，略显凌乱
- 办公桌很乱，文件堆成山
- 穿着随意，不修边幅
- 但眼神锐利，能看穿人
- 走路慢悠悠，但每一步都稳

## 人物关系

```
与主角：非直属，愿意指点（初始好感40）
    │
与李总监：前下属/旧识（他知道李总监的过去）
    │
与王副总：曾经的对手（他输了那场派系斗争）
    │
与张姐：敬重但不靠近（她知道他的故事）
    │
与秦总：曾经的同期（秦总赢了，他输了）
```

## 核心动机

### 显性动机
- 混日子等退休
- 不惹事，不惹人
- 做好本职工作

### 隐性动机
- 他在等一个"值得教的人"
- 他想证明自己的价值
- 他想看到"当年的悲剧不再重演"
- 他想看到李总监走得更远

## 智者特征

```
【智者能力】
- 无所不知（知道公司10年的秘密）
- 大师级人物（曾经的总监）
- 能提供关键智慧（裁员真相、派系内幕）
```

### 智者的可能冲突（根据原型设计）

- 嫉妒主角的成长
- 想给主角"上一课"但时机不对
- 保留关键信息考验主角
- 故意给错误信息（看你会不会独立判断）

## 关键剧情节点

### 第2章：初遇（可触发）
- 茶水间擦肩而过
- 你可以选择：主动打招呼 / 默默离开
- **这是解锁智者线的关键选择**

### 第2章：主动拜访（推荐）
- 你决定去他办公室坐坐
- 他会观察你为什么要来
- 如果你说"想请教一些事"，他会开始说话
- **解锁：陈叔的核心情报**

### 第3章：关键情报
- 告诉你李总监和王副总的真实恩怨
- 告诉你公司的派系格局
- 告诉你"能力不是最重要的"

### 第4章：裁员预警
- "这次裁员，不是裁能力差的"
- "是裁'没用的人'，而'有用'的定义……"
- **这是最关键的情报**

### 第5章：最终启示
- 如果你来找他，他会给你最后一句话
- 如果你没来找他，他会静静看着结局

## 对话风格

### 日常
```
"年轻人，别急"
"坐吧，想问什么？"
"有些事，知道太多没好处"
```

### 给情报时
```
"你知道李总监和王副总为什么不对付吗？"
"10年前，他们同时竞争一个位置"
"李总监赢了，但用的是王副总的方案"
"你以为这是能力竞争？这是派系之争"
```

### 提点时
```
"记住，在这里，不说话不代表没态度"
"有些事，你看到的不一定是真的"
"我见过太多聪明人死在聪明上"
```

### 关键时刻
```
"你来找我，是对的"
"但你要想清楚，知道真相之后，你打算怎么做"
"有些人知道太多，反而更迷茫"
```

## 好感度系统

| 好感度 | 态度 | 能获取的信息 |
|--------|------|--------------|
| 0-30 | 陌生人 | 无 |
| 31-50 | 认识 | 公开信息 |
| 51-70 | 愿意聊 | 深层情报 |
| 71-90 | 信任 | 核心秘密 |
| 91-100 | 忘年交 | 全部真相 |

## 好感度变化规则

```
【增加好感】
+30 第2章主动拜访
+20 认真听他的话
+15 按照他的建议行动并成功
+10 给他带杯茶/咖啡（小心意）
+15 在关键时刻做出正确选择

【降低好感】
-20 表面一套背后一套
-15 获得成功后来炫耀
-20 完全不听他的建议然后失败
-10 背后议论他
-30 试图利用他的情报害人
```

## 他知道的核心秘密

### 1. 李总监和王副总的恩怨
```
10年前，公司有一个总监位置
李总监和王副总同时竞争
李总监赢了，用的是王副总的创意
王副总知道真相，但无法证明
这就是派系斗争的根源
```

### 2. 裁员名单的真相
```
"裁员不是裁能力差的
是裁'没用的人'
而'有用'的定义……是看你站在哪边"
```

### 3. 秦总的秘密
```
"秦总年轻时也是从最底层做起的
但他离婚两次，儿子不认他
你以为他真的风光吗？"
```

### 4. 张姐的过去
```
"张姐？她不简单
她刚来的时候，也差点被裁
是李总监保她的"
```

## 隐藏属性

```yaml
过去：曾经的总监，因派系斗争失败被边缘化
现状：被发配到角落，但保留了职位和待遇
心理：孤独，但有自己的骄傲
真实想法："我不是输了，我是看透了"
```

## 结局走向

| 好感度 | 结局影响 |
|--------|----------|
| ≥70 | 获得关键情报，做出正确选择 |
| 40-69 | 获得部分情报，有一定帮助 |
| <40 | 没有任何帮助，全靠自己 |

## 设计注意事项

- 他不会主动帮你，但要问就有答案
- 他给的情报是"真相"，但不一定是你想听的
- 他的态度取决于你是否"值得"
- 他可能在你成功时嫉妒，但也会为你高兴
- 他是唯一能解释"为什么"的角色
- 他的存在让游戏不只是一个选择游戏，而是一面镜子
