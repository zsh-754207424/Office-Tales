# 小刘HR - 超能力者原型

## 基础信息

| 字段 | 内容 |
|------|------|
| **姓名** | 刘思琪 |
| **职位** | HR专员 |
| **年龄** | 26岁 |
| **性格** | 热情、公事公办、嘴严、细心 |
| **原型** | 超能力者（Wizard/Oracle） |
| **出场** | 第2章（入职手续） |

## 外表印象

- 162cm，标准职业装
- 笑容客气但保持距离
- 说话语速不快，但信息量大
- 随身带着笔记本，记录一切
- 看起来普通，但眼神锐利

## 人物关系

```
与主角：公事公办（初始好感55）
    │
与所有部门：保持中立
    │
与领导层：知道所有人事变动
    │
与陈叔：有限度提供信息
```

## 核心动机

### 显性动机
- 做好本职工作
- 按流程办事
- 不出错

### 隐性动机
- 她知道很多秘密，但不能说
- 她有自己的职业道德
- 她在等一个机会（升职/转岗）

## 超能力者特征

```
【超能力】
- 知道所有人事变动
- 知道裁员名单
- 知道所有人的薪酬和绩效
- 知道领导的真实评价

【限制】
- 职业道德让她不能说
- 她只能用暗示的方式
- 说得太明显会丢掉工作
```

## 关键剧情节点

### 第2章：入职手续
- 办理入职、签合同
- 这是第一次接触
- 她会观察你这个新人

### 第3章：政策咨询
- 你可以向她打听公司政策
- 她会给你官方信息
- 但不会给你"内幕"

### 第4章：裁员预警（关键）
- 她会提前给你暗示
- "最近部门变动比较多"
- "你要做好心理准备"

### 第5章：最终时刻
- 裁员当天
- 她负责办理手续
- 她的态度取决于你们的关系

## 对话风格

### 官方回复
```
"这个我需要核实一下再回复你"
"公司流程是这样的……"
"我可以帮你查一下"（但查不查不一定）
```

### 暗示时
```
"最近……部门可能要有些调整"
"你要做好心理准备"
"我说的你懂的"
```

### 高好感时
```
"名单上有你"
"还有3天，你可以想想办法"
"我可以帮你争取多一周"
"内部调岗的窗口还开着"
```

### 拒绝时
```
"这个我不清楚"
"你需要问你的领导"
"流程上是这样的，我没有权限"
```

## 好感度系统

| 好感度 | 态度 | 能获取的信息 |
|--------|------|--------------|
| 0-40 | 公事公办 | 官方信息 |
| 41-60 | 客气友好 | 部分暗示 |
| 61-80 | 愿意帮忙 | 核心情报 |
| 81-100 | 私下帮忙 | 最大支援 |

## 好感度变化规则

```
【增加好感】
+10 礼貌沟通
+15 尊重她的工作
+10 不问敏感问题
+15 在关键时刻保持体面
+20 她的好意你记得

【降低好感】
-10 态度傲慢
-15 问太敏感的问题
-20 让她为难
-15 在裁员时失控/纠缠
-25 试图贿赂/套近乎（她很反感）
```

## 她掌握的核心情报

### 1. 裁员名单
```
她知道最终名单
但不能说
只能说暗示
```

### 2. 薪酬绩效
```
所有人的薪酬级别
绩效考核结果
晋升提名名单
```

### 3. 领导评价
```
领导对每个下属的真实评价
（不一定和表面一致）
```

### 4. 内部调岗
```
哪些部门有空缺
哪些岗位在招人
调岗流程是什么
```

## 暗示系统设计

### 好感度 < 40
```
问：最近部门有什么变化吗？
答："这个我不清楚，你需要问你的领导"
```

### 好感度 40-60
```
问：最近部门有什么变化吗？
答："最近确实变动比较多，你自己注意一下"
```

### 好感度 61-80
```
问：最近部门有什么变化吗？
答："名单下周出，你……做好心理准备"
```

### 好感度 > 80
```
问：最近部门有什么变化吗？
答："名单上有你。但还有3天，你可以想想办法"
```

## 隐藏属性

```yaml
职业操守：很强，不会主动透露机密
但有例外：如果你真的值得帮，她会冒一点险
真实想法："我能做的就这么多，剩下的靠你自己"
生存法则：在HR的位置上，知道太多不能说太多
```

## 结局走向

| 好感度 | 结局影响 |
|--------|----------|
| ≥80 | 提前告诉你结果，帮你争取缓冲时间，介绍其他公司岗位 |
| 50-79 | 正常办理手续，不会为难你 |
| <50 | 照章办事，不提供任何帮助 |

## 设计注意事项

- 她不是你的盟友，是公司的员工
- 她给的信息都是"暗示"，不会直接说
- 她的帮助是有限度的，不会冒着丢工作的风险
- 她是游戏中的"情报源"，但获取情报需要好感度
- 她的存在让玩家知道"信息不对称"是职场现实
- 在裁员时刻，她的态度会直接影响玩家的感受
