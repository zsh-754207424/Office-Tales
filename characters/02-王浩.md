# 王浩 - 劲敌原型

## 基础信息

| 字段 | 内容 |
|------|------|
| **姓名** | 王浩 |
| **职位** | 专员（与主角同期入职） |
| **年龄** | 26岁 |
| **性格** | 表面热情、城府深、野心大、表演型人格 |
| **原型** | 劲敌（Rival） |
| **出场** | 第1章（入职第一天） |

## 外表印象

- 178cm，阳光帅气，爱笑
- 笑容很标准，像是练习过无数遍
- 说话圆滑，从不得罪人
- 永远一副"我是好人"的样子
- 但笑容不达眼底

## 人物关系

```
与主角：同期入职的竞争对手（初始敌意50）
    │
与王副总：暗中投靠（被招揽/利用）
    │
与李总监：表面服从，暗中使绊
    │
与张姐：表面友好（试探她能不能被拉拢）
    │
与小林：笼络新人（建立自己的小团队）
```

## 核心动机

### 显性动机
- 在公司站稳脚跟
- 获得晋升
- 证明自己不比别人差

### 隐性动机
- 他家里有重病的母亲，需要钱
- 他曾经也是被欺负的新人，被逼到墙角过
- 他不相信"凭实力"能赢，必须用手段

## 劲敌特征

```
【经典劲敌模式：纽曼 vs 杰瑞】
- 表面和谐，背后对抗
- 不会正面冲突
- 总在关键时刻出手
- 仇恨主角但没有恶意（他的恶意是对事不对人）
```

## 行为模式

### 表面行为
```
"我们是好兄弟嘛！"（然后背后捅刀）
"这个我帮你做吧"（然后功劳归他）
"领导对你印象不错啊"（试探你）
"我觉得你做得挺好的啊"（捧杀）
```

### 暗中行为
```
- 向领导打小报告（你的负面信息）
- 在同事中散布不利消息
- 关键时刻踩你上位
- 把你的方案"优化"后汇报
```

## 关键剧情节点

### 第1章：同期入职
- 你们同时入职，天生竞争关系
- 他主动示好，试探你的性格
- 你选择：接受示好 / 保持距离

### 第2章：抢功事件（核心冲突）
- 你做了一个方案/报告
- 他"顺手"拿去给领导看
- 领导表扬了他，你被忽视
- **这是你和他的第一个重大冲突**

### 第3章：晋升竞争
- 有一个晋升名额
- 你们同时竞争
- 他可能暗中搞破坏
- 正面竞争 vs 暗中使绊子

### 第4章：裁员危机
- 裁员名单二选一
- 你们都知道只有一个人能留下
- 他会做什么？

### 第5章：最终对决
- 必须有一个结果
- 你们的关系无法和解（设计如此）
- 无论谁赢，另一个都不会消失

## 对话风格

### 表面友好
```
"早啊！今天状态不错嘛"
"这个我帮你看了一下，有个地方可能要注意……"
"我们是一期的，要互相帮衬啊"
```

### 暗中试探
```
"听说你昨天加班到很晚？领导知道吗？"（暗示你效率低）
"张姐对你挺好的啊，有什么秘诀吗？"（试探你的关系网）
"王副总今天好像心情不好，你小心点"（给你情报？但可能误导你）
```

### 撕破脸时（罕见）
```
"别装了，我知道你看我不顺眼"
"职场就是这样，你不踩人人就踩你"
"你以为你赢了？还早呢"
```

## 敌意值系统

| 敌意值 | 态度 | 行为 |
|--------|------|------|
| 0-30 | 表面友好 | 和你套近乎 |
| 31-50 | 暗中观察 | 寻找你的弱点 |
| 51-70 | 主动出击 | 背后搞小动作 |
| 71-90 | 公开对抗 | 正面竞争不择手段 |
| 91-100 | 不死不休 | 拼个你死我活 |

## 敌意值变化规则

```
【增加敌意】
+20 抢功事件被发现
+15 你获得领导表扬
+10 你获得重要机会
+20 你公开质疑他
+15 张姐站在你这边

【降低敌意】
-5 你主动示好（但他可能更警惕）
-10 你选择退让（他觉得你好欺负）
-15 你在竞争中输给他（他赢了会放松）
```

## 隐藏属性

```yaml
家庭情况：母亲尿毒症，每周透析3次
经济压力：需要钱，必须保住这份工作
心理状态：焦虑症前期，靠药物控制
真实恐惧：害怕失败，害怕回到当初被欺负的日子
```

## 结局走向

| 结果 | 描述 |
|------|------|
| **你赢** | 他被裁，走向"独狼"路线，可能跳槽后成为对手 |
| **他赢** | 你被裁，走向"复仇者"路线（如果有续作） |
| **平手** | 两人都留，关系彻底破裂，但无法分出胜负 |

## 设计注意事项

- 他不是纯粹的坏人，有自己的苦衷
- 他对主角的仇恨来源于"你威胁到了他"
- 他不会主动伤害你，但会自保时踩你
- 无论结局如何，他都会继续存在（劲敌特质）
- 玩家无法和他和解，这是设计的一部分
- 隐藏选项：如果你在某节点击败他，他可能从敌意转为"敬畏"
