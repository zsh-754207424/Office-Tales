## 经历塑造

### 催化剂事件

```
【推动改变的催化剂】

- 3年前，王浩刚入职时，被人抢功
- 他当时站在旁边，没有说话
- 王浩的眼神让他想起当年的自己

事件2：儿子的电话
- 半年前，儿子打电话说不想见他
- 他第一次失眠了整晚
- 开始反思自己的选择

事件3：陈叔的话
- 1年前，陈叔对他说：
  "老李，我们都是失败的父亲"
- 这句话刺痛了他很久
```

### 深度认识

```
【认识他人阶段】

他认识到的：
- 张姐的忠诚是有条件的
- 王浩的野心是写在脸上的
- 小林（主角）的迷茫像极了当年的自己

【认识自我阶段】

他认识到的：
- 他把对儿子的愧疚投射到了下属身上
- 他害怕失去，所以不敢信任
- 他用工作麻痹自己

【承担个人责任】

他决定：
- 给小林机会，不是因为小林，是因为想弥补当年的自己
- 如果有一天见到儿子，要道歉
```

### 行动

```
【基于认识的行动】

行动1：对小林的态度
- 表面：严厉要求
- 实际：希望小林比自己更强

行动2：对儿子的弥补
- 每个月给儿子写信（从不寄出）
- 攒钱给儿子出国留学
- 开始每周给儿子打电话

行动3：对自己的改变
- 减少安眠药用量
- 开始运动
- 接受自己不是完美父亲
```
