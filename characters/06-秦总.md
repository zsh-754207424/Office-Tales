# 秦总 - 翻版原型

## 基础信息

| 字段 | 内容 |
|------|------|
| **姓名** | 秦志远 |
| **职位** | 总经理 |
| **年龄** | 48岁 |
| **性格** | 不怒自威、高瞻远瞩、深不可测 |
| **原型** | 翻版（Threshold Guardian / Role Model） |
| **出场** | 第3章（季度大会远观）/ 第5章（结局对话） |

## 外表印象

- 180cm，身姿挺拔，略有发福
- 头发花白但精神矍铄
- 永远西装，偶尔不系领带
- 说话不多，但每句话都有分量
- 全场会议他说10分钟，但没人敢不听

## 人物关系

```
与主角：终极目标（你永远到不了的位置）
    │
与李总监：提拔/观察（秦总在看他培养的人如何）
    │
与王副总：制衡/观察（秦总在利用派系斗争）
    │
与陈叔：曾经的同期（秦总赢了，陈叔输了）
    │
与所有人：最高权力，无人敢挑战
```

## 核心动机

### 显性动机
- 公司发展
- 业绩增长
- 稳定大局

### 隐性动机
- 培养下一代接班人
- 维持派系平衡
- 确保自己退休后的安排

## 翻版特征

```
【翻版的定义】
- 主角想成为的人
- 故事的终极目标
- 代表成功的代价

【秦总的地位】
- 你能达到的最高点
- 你永远无法直接对话（直到结局）
- 他的存在是一个标杆
```

## 关键剧情节点

### 第1章：传说
- 茶水间闲聊提到他
- "听说秦总年轻时也是从最底层做起的"
- "他每天6点到公司，现在也一样"

### 第3章：首次远观
- 季度总结大会
- 你第一次看到他
- 他全程只说了10分钟
- 但全场鸦雀无声

### 第4章：侧面影响
- 裁员决定最终要他点头
- 他的一个决定影响所有人
- 你永远不会知道他的想法

### 第5章：结局对话（如晋升）
- 你终于有机会和他单独说话
- "你就是李总监说的那个年轻人？"
- 这是你们第一次，也是最后一次对话

## 对话风格

### 大会上
```
"我不听过程，我只看结果"
"能者上，庸者下"
"公司不养闲人"
（简短，但没人敢反驳）
```

### 私下（结局时）
```
"年轻人，你做得不错"
"但这才是开始"
"你看到的是我的位置，你看不到的是……"
（然后不说了）
```

### 隐藏面
```
（你永远不会直接知道）
- 他离婚两次
- 他儿子不认他
- 他的身体已经有问题
- 他的压力比你大一百倍
```

## 他的态度系统

```
【他观察的内容】
- 李总监培养的人如何
- 各派系的平衡
- 谁能担大任

【他不在乎的内容】
- 个别员工的小心思
- 办公室政治细节
- 短期利益
```

## 隐藏属性

```yaml
家庭：离婚两次，儿子在国外不联系
健康：高血压、糖尿病前期
睡眠：靠药物，每天只睡4-5小时
真实想法："谁能接我的班？"
代价：他得到了一切，但失去了很多
```

## 结局选择

### 第5章结局选项
```
秦总私下问你："你想成为我吗？"

A. "是的，我想"
   → 你接受了这条路的代价
   → 走向"权力路线"

B. "不完全是"
   → 你有自己的人生目标
   → 走向"平衡路线"

C. "我不知道"
   → 你还没想清楚
   → 走向"探索路线"
```

## 设计注意事项

- 他是"成功"的象征，但不代表完美
- 他的存在让玩家思考"成功值得吗？"
- 他不会直接帮助或伤害你，但你在他观察的范围内
- 他的隐藏面（代价）只有在结局时才会揭示
- 他代表的是"终点"，但这个终点不一定是最好的
- 玩家在结局时有机会评价他，也评价自己
