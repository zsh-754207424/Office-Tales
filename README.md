## 项目简介

这是一个职场剧情互动小说设计框架，包含完整的剧情结构、角色设定和多分支故事。

你将扮演一名职场新人，在派系斗争中做出选择，最终决定自己的职场命运。

---

## 核心特色

### 📖 5章完整剧情设计
基于 BS2（Blake Snyder Beat Structure）节拍表设计，覆盖：
- 入职适应 → 首个任务 → 团队协作 → 利益冲突 → 结局

### 👥 9个深度角色原型
每个角色包含：
- 表面生活（外表、工作、习惯）
- 内心生活（动机、恐惧、信念）
- 成长弧线（入职前 → 辉煌期 → 现在）

### 🎯 24种可能结局
- A类结局：晋升
- B类结局：稳定留下
- C类结局：离职
- 组合变量：价值观选择 × 派系站队 × 关键决策

### 🎭 冲突驱动叙事
应用罗伯特·麦基的对白理论：
- 均衡冲突 / 非直接冲突 / 自反性冲突
- 每段对话都有戏剧张力

### 🕵️ 悬念制造系统
12种悬念方法：
- 滴答时钟 / 悬而未决 / 戏剧反讽
- 延长悬念技巧让读者欲罢不能

## 核心方法论

### BS2 节拍表

| 幕 | 内容 |
|---|------|
| 第一幕（30%） | 建置 → 对应 → 狂欢 → 第二扇门 |
| 第二幕（50%） | B故事 → 娱乐游戏 → 中点 → 坏蛋逼近 → 一无所有 → 灵魂黑夜 |
| 第三幕（20%） | 第三幕衔接点 → 最终对决 → 结局 |

### LOCK 系统
- **L**ead（主角）：有缺陷但值得关注的角色
- **O**bjective（目标）：明确的目标驱动故事
- **C**onfrontation（冲突）：层层升级的障碍
- **K**nockout（结局）：满足收尾

### 七种冲突类型
| 冲突类型 | 适用场景 |
|---------|---------|
| 均衡冲突 | 势均力敌的对抗 |
| 喜剧冲突 | 幽默场景需要深度 |
| 不对称冲突 | 权力不对等 |
| 非直接冲突 | 通过第三方表达 |
| 自反性冲突 | 角色内心矛盾 |
| 微限冲突 | 微小细节中的张力 |
| 混合型 | 复杂场景 |
