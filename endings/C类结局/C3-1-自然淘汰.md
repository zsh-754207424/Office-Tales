## 场景：自然淘汰

```
这不是你的错。

也许你不适合这里。
也许这里不适合你。

名单公布的那天，你反而松了一口气。

你坐在小刘HR的办公室里，签着各种文件。

小刘看着你，表情有些复杂：

"小林……你有没有想过，也许这里不适合你？"

你沉默。

她继续说：

"你不是能力不行。
你是不适合这种环境。

有些人适合在狼群里生存。
有些人适合在草原上奔跑。

你是后者。"
```

---

## 内心独白

```
我付出了时间。但我也认清了自己。

我不是能力不行。我只是不适合这里。

这……也是一种答案。

有时候，知道不适合什么，
比知道适合什么更重要。

我不是被淘汰了。
只是被放错了位置。
```

| 代价 | 获得 |

## 角色对话

```
李总监：

但这不代表你不好。"

```

---

## 终场画面

```
你走出公司。没有回头。

你想起入职第一天，你站在楼下，紧张得手心出汗。

现在你知道了。

你适合……更好的地方。
更适合你的地方。

不是这里不够好。
是你们不合适。
```
