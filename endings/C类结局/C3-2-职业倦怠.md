| 稀有度 | 2% |

---

## 场景：职业倦怠

```
你撑不住了。

不是因为能力。
是心态。

每天早上，你都要在床上躺很久才能爬起来。
你不想去公司。你不想看见那些人。
你不想再做那些事。

你累了。不是身体上的累。
是心累。

你去看了心理医生。

医生看着你的测评结果，沉默了很久：

"你需要休息。"

他推给你一张假条：
"我给你开一个月的。你先休息。"

你拿着假条，不知道该高兴还是该难过。
```

---

## 内心独白

```
我付出了健康的代价。
但我也学会了照顾自己。

张姐说得对——有时候，停下来也是一种前进。

我以为坚持就是胜利。
但有些时候，坚持……只是慢性自杀。

我需要休息。
我需要重新开始。

这不是失败。
这是……自我救赎。
```

---

## 代价与获得

| 代价 | 获得 |
|------|------|
| 付出了健康的代价 | 健康意识 |
| 心态崩溃 | 学会休息 |
| 职业中断 | 重新审视人生 |

---

## 角色对话

```
张姐：
"你太拼了。
有时候，停下来也是一种前进。"

心理医生：
"你把自己逼得太紧了。
你不是机器。"
```

---

## 终场画面

```
你走出公司。没有拿纸箱——
你的东西还在工位上。

但你不会回去了。

你走在大街上，阳光照在你身上。
你第一次感觉轻松了。

也许……休息一下，真的没什么不好。

你不是机器。
你是一个人。
```

---

> **职业倦怠 — 你的选择，决定了你的职场命运。每一个选择都有代价，每一个代价都是成长。**
