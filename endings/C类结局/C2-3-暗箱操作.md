---

## 场景：暗箱操作

```
你做了一些违背良心的事。

也许是为了生存。
也许是为了上位。

但当你做完那些事后，你无法面对镜子里的自己。

那天晚上，你躺在床上，睁着眼睛看着天花板。

你想起陈叔说的话：
"不要走我的老路。"

你想起小林看你的眼神——那种信任。

你想起入职第一天的自己——
那个什么都害怕但什么都想做好的人。

你坐起来。
你做了一个决定。

离开这里。
重新开始。

希望……还不算太晚。
```

---

## 内心独白

```
我选择了离开。
不是因为失败。
是因为无法面对这样的自己。

我做了不对的事。我知道。
但我还有机会改正。

离开这里。重新开始。

希望……还不算太晚。

至少我还有良心。
这说明……我还没有完全失去自己。
```

---

## 代价与获得

| 代价 | 获得 |
|------|------|
| 你选择了离开 | 良心 |
| 不是因为失败，而是因为无法面对 | 自省 |
| 面对自己的错误 | 重新开始的机会 |

---

## 角色对话

```
陈叔：
"你是个有良心的人。离开是对的。"

（你看着陈叔，他也在看着你。
你们两个都知道——
留在这里，你会变成他。
离开，是你最后的机会。）
```

---

## 终场画面

```
你走出公司。手里的纸箱很轻，但你的心很重。

你知道自己做了什么。
但你也知道自己还有机会。

这一次……
你不做违心的事了。

哪怕前面更难走。
但至少……你还能面对自己。
```
