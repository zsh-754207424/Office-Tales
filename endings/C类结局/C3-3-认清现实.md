## 场景：认清现实

```
你认清了这里的本质。
你发现这里不适合你。

不是失败。是选择。

在你离开之前，你去找了陈叔。
你坐在他办公室里，喝着凉了的茶。

陈叔看着你：
"你想清楚了？"

你点头。

他笑了笑：
"好。"

他站起来，走到窗边：
"你能看清这里，说明你有智慧。
希望你能找到真正适合你的地方。"

他转过身：
"你还年轻。有的是机会。"
```

---

## 内心独白

```
我认清了现实。但我也保持了初心。
我开始了新的旅程。

我不后悔来这里。
因为这三个月让我明白了一件事：
有些地方，你不适合，不是因为你有问题。
而是因为……你选择了另一条路。

那条路不一定更轻松。
但那是你自己的路。
```

```
陈叔：

希望你能找到真正适合你的地方。"

（他拍了拍你的肩膀。

他也看清了这里。
但他走不了。
你还可以。）
```

---

## 终场画面

```
你走出公司。回头看了一眼这栋大楼。

三个月的经历，像一场梦。

但你现在醒了。

你往前走。

这一次，你知道自己要什么了。
至少……你知道自己不要什么了。
```
