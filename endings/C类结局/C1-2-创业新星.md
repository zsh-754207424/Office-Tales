## 场景：创业新星

```
你选择了创业。
不是被裁员，是主动出走。

你在公司学到的一切——
产品知识、客户资源、管理经验——
现在都变成了你的资本。

你约了三个前同事，在一家咖啡馆里聊了一整个下午。
你们决定一起做点什么。

李总监听说你要创业，没有拦你。
他把你叫到办公室，给了你一张名片：

"这是我认识的一个投资人。如果你需要……"

你接过名片，有些意外。

李总监难得地笑了：
"年轻人有闯劲，是好事。"
```

---

## 内心独白

```
我选择了不确定性。
但我也选择了自由。

创业九死一生。这我知道。
但留在这里……也是一样的。

在职场，你永远不知道明天的名单上有没有你。
在创业，至少命运掌握在自己手里。

至少……我试过。
```

---

## 代价与获得

| 代价 | 获得 |
|------|------|
| 你选择了不确定性 | 自由 |
| 放弃了稳定收入 | 控制自己的命运 |
| 面对失败的风险 | 实现梦想的机会 |

## 角色对话

```
李总监：
"创业九死一生，你想清楚了吗？
但年轻就是资本，去吧。"

小林（电话）：
"前辈，听说你创业了……
我……我能跟你一起干吗？"
```

---

## 终场画面

```
你拿着李总监给的名片，站在公司楼下。
阳光很刺眼。但你感觉前所未有的清醒。

前方是未知的路。
但你不再害怕了。

因为你知道——
害怕也没用。
去做就行了。
```
