# C2-2：被迫离开

---

## 结局条件

| 项目 | 内容 |
|------|------|
| 结局编号 | C2-2 |
| 标题 | 被迫离开 |
| 解锁条件 | 中立失败，两边都针对，被迫离职 |
| 稀有度 | 4% |

---

## 场景：被迫离开

```
你试图保持中立。但中立在这里行不通。

当两边都在拉拢你的时候，你选择了"不选"——

这在派系斗争的词典里，就是"背叛了两边"。

你成了两边的眼中钉。
李总监不再给你重要的工作。
王副总开始散布关于你的"传闻"。

你坐在工位上，感觉四面都是墙。

你最后主动递交了辞职信。
比被裁好一点。
至少……你保住了尊严。
```

---

## 内心独白

```
中立不是出路。你必须做出选择。

但选择……意味着放弃一部分自己。

我保持了中立。但我失去了工作。

这值得吗？

我不知道。
但我知道，我学会了：
有些地方，没有中立的选择。
不选，也是一种选——而且可能是最差的那种。
```

---

## 代价与获得

| 代价 | 获得 |
|------|------|
| 中立不是出路 | 认清现实 |
| 你失去了工作 | 保持自己的清白 |
| 被迫离开 | 知道中立的风险 |

## 角色对话

```
小刘HR：

```

---

## 终场画面

```
你拿着离职证明，走出公司。你没有回头。

你知道，这个教训会跟着你很久。

下一次……
你还会中立吗？
```

---

> **被迫离开 — 你的选择，决定了你的职场命运。每一个选择都有代价，每一个代价都是成长。**
