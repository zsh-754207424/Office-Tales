|------|------|

| 标题 | 派系牺牲品 |
| 解锁条件 | 站错队，被清洗，被迫离开 |
| 稀有度 | 5% |

---

## 场景：派系牺牲品

```
你选择了错误的一边。

派系斗争结束了。你站的那一边输了。
输得很彻底。

名单公布的那天，你的名字在上面。

你坐在工位上，看着同事们来来往往。
没有人看你。
你像是已经不存在了一样。

王副总从你身边经过。
他看了你一眼，那眼神里没有任何温度。

然后他走了。
就像你从来没有为他做过任何事一样。

你为他做了那么多。
他连一句话都没有替你说。
```

---

## 内心独白

```
我付出了代价。我站错了队。

这不是能力的失败。是站队的失败。

职场就是这样。

下次……我会选得更聪明一些。
或者……不再参与这个游戏。

我学到了一课——
站队的时候，你以为你选了一个靠山。
但其实，你只是选了一块墓碑。
```

---

## 代价与获得

| 代价 | 获得 |
|------|------|
| 付出了代价 | 教训 |
| 站错队 | 看清职场本质 |
| 被迫离开 | 成长 |

---

## 角色对话

```
王副总：
"职场就是这样。
下次，选择要更聪明一些。"

小刘HR：
"这是你的离职手续。
下次……不要太轻信别人。"
```

---

## 终场画面

```
你抱着纸箱，走出公司。夕阳照在你身上。
但你感觉不到温暖。

你输了。
但你还活着。

还有下一次。
下一次……你会选对的？
还是……不再选？
```

---

> **派系牺牲品 — 你的选择，决定了你的职场命运。每一个选择都有代价，每一个代价都是成长。**
