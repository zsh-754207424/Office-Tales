# C1-1：优雅转身

---

## 结局条件

| 项目 | 内容 |
|------|------|
| 结局编号 | C1-1 |
| 标题 | 优雅转身 |
| 解锁条件 | 李总监信任≥70，主动选择离开，高姿态 |
| 稀有度 | 4% |

---

## 场景：优雅转身

```
你主动选择了离开。

不是被裁员，不是因为失败。
是因为你找到了更好的机会。

你在公司的最后一天。
李总监把你叫到办公室。

他递给你一个信封——里面是一封推荐信。

"你做得很好。"
他看着你，眼神里有不舍，也有尊重：
"如果你以后有需要，可以找我。"

你接过信封，点了点头。

你走出李总监的办公室。
最后看了一眼这个你待了三个月的地方。

你在这里哭过、笑过、愤怒过、迷茫过。
但现在，你带着尊严离开。
```

---

## 内心独白

```
我选择了新的开始。
但我也离开了熟悉的一切。

我不后悔。
这里的三个月让我明白了一件事：
有些地方，你注定只是过客。

不是这里不好。
不是我不够好。
只是……不合适。
```

---

## 代价与获得

| 代价 | 获得 |
|------|------|
| 你选择了新的开始 | 新机会 |
| 离开了熟悉的一切 | 尊严 |
| 开始新的故事 | 李总监的尊重 |

---

## 角色对话

```
李总监：
"你有更好的机会，我理解。
如果以后有需要，可以找我。"

陈叔：
"你走得比我想象的更远。
你比我有勇气。"
```

---

## 终场画面

```
你拎着纸箱，走出公司大门。夕阳照在你身上。

你回头看了一眼这栋30层的写字楼。

三个月前，你从这里开始。
三个月后，你在这里结束。

但这不是终点。是起点。
```

---
