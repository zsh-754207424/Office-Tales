## C类结局总体主题

> 离开不是失败。有时候，放手也是一种勇气。

C类结局的共同特征是"离开"——你不再属于这家公司。但离开的方式和原因各不相同。

每一个C类结局，都在说同一句话：**不是每个地方都适合每个人。离开，有时候是最好的选择。**
