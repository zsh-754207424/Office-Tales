# C1-3：回归家庭

---

## 场景：回归家庭

```
你选择了停下来。

不是失败。是你找到了更重要的东西。

你在公司的最后一天，给你妈妈打了一个电话。

"妈，我离职了。"

电话那头沉默了几秒。

然后你妈妈说：
"回来吧。妈给你做饭。"

你挂了电话，哭了。

这是三个月来，你第一次哭。

不是因为失败。
是因为你终于可以放松了。
```

---

## 内心独白

```
我选择了家庭。
但我也放弃了职场的机会。

但……我找到了内心的平静。

李总说，有时候放下也是一种选择。
以前我不理解。
现在……我明白了。

人生不只是职场。
还有家人。还有自己。
```

---

## 代价与获得

| 代价 | 获得 |
|------|------|
| 你选择了家庭 | 家庭 |
| 放弃了职场的机会 | 内心的平静 |
| 职业中断 | 自我 |

---

## 角色对话

```
李总监：
"有时候，放下也是一种选择。
希望你能找到你想要的。"

妈妈：
"回来吧。妈给你做饭。
你瘦了。"
```

---

## 终场画面

```
你收拾好东西，走出公司。
这一次，你没有回头。

你要回家。

家里有妈妈做的饭。
有你的房间。
有你的过去。

也许还有……你的未来。
```

---

> **回归家庭 — 你的选择，决定了你的职场命运。每一个选择都有代价，每一个代价都是成长。**
