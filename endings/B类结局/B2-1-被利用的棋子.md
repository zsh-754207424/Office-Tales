| 解锁条件 | 中立派系，被两边利用，完成任务 |
| 稀有度 | 8% |

---

## 场景：被利用的棋子

```
你是夹在派系之间的人。

李总监的人来找你帮忙——你帮了。
王副总的人也来找你帮忙——你也帮了。

你以为这是"人缘好"。

直到有一天，你发现两边都在用你——

你帮李总监做的事，被用来打击王副总。
你帮王副总做的事，被用来攻击李总监。

你是一个工具。
两边都在用你。
但你两边都不属于。

你是那个在棋盘上走来走去的棋子。
哪里需要你，你就去哪里。
但没有人问你愿不愿意。
```

---

## 内心独白

```
我的价值被消耗了。
我做了很多事，但没有人为我说话。

但我也学会了在夹缝中生存。
我开始寻找自己的路。

也许有一天，我能跳出这个棋盘。
不再是棋子，而是下棋的人。

但那天……还没到。
```

---

## 代价与获得

| 代价 | 获得 |
|------|------|
| 你的价值被消耗了 | 生存能力 |
| 做了很多事没有人替你说话 | 看清现实 |
| 两边不属于 | 学会自我保护 |

---

## 角色对话

```
李总监：
"你是个可用的人。
但你要有自己的立场。"

张姐：
"在这里，好人是活不长的。
你太好说话了。"
```

---

## 终场画面

```
你坐在工位上，看着两边的人在走来走去。

你学会了一件事：
在这里，你不能只是"好人"。

你有自己的路要走。
你不能永远做一颗棋子。
```
