## 场景：夹缝生存

```
你选择了最安全的方式。
不站队，不惹事，保持中立。

你对每个人都笑。对每个人都客气。

李总监安排的任务你做。
王副总安排的"帮忙"你也做。

你以为这样就能平安活下来。

但有一天，小刘HR在走廊里叫住你：

"小林，你知道为什么你这次考评是C吗？"

你摇头。

"因为没有人给你说话。
你没有站队，所以没有人把你当自己人。
没有人把你当自己人，所以没有人愿意帮你说话。"
```

---

## 内心独白

```
我活下来了。但没有归属感。

我可能是安全的。
但也是孤独的。

中立是对的吗？
如果中立是对的，为什么我失去了所有保护？
```

---

## 代价与获得

| 代价 | 获得 |
|------|------|
| 活下来了，但没有归属感 | 安全 |
| 可能是安全的，但也可能是孤独的 | 没有树敌 |
| 开始思考中立是对的吗 | 独立 |

## 角色对话

```
张姐：
"你是个聪明人。

```

## 终场画面

```
你回到工位。

桌上有一个便签，上面是你自己写的——
"中立。"

你看着那两个字，觉得它们很可笑。

中立？在这里……
中立就是没有保护伞。
```
