## 场景：稳健派

```
你留下来了。没有晋升，没有加薪。
只是……留下来了。

你继续做你的工作。
每天准时上班，准时下班。
偶尔加班，偶尔偷懒。

你的工位上贴着一张便签，上面写着：
"不要犯错。"

你还和以前一样——不冒尖，不落后。
你在中间位置，安全地待着。

这就是你的职场人生。

没有大风大浪。
也不会有什么惊喜。
```

---

## 内心独白

```
这是大多数人的结局。
我选择了稳定，放弃了冒险。

我可能永远不会有大的成就。
但我也不会摔得很惨。

这是……妥协吗？还是……成熟？

也许这就是大多数人的职场人生。
平庸，但安全。
```

---

## 代价与获得

| 代价 | 获得 |
|------|------|
| 选择了稳定，放弃了冒险 | 稳定 |
| 可能有遗憾，但也不后悔 | 平静 |
| 学会了和自己和解 | 安全 |

---

## 角色对话

```
李总监：
"你做得不错。继续保持。"
```

---

## 终场画面

```
你收拾好东西，准备下班。今天没有加班。

你走出公司大楼，夕阳照在你身上。

这样的日子，还要过很多年。
但至少……你还在这里。

这样就够了？
也许吧。
```

---

> **稳健派 — 你的选择，决定了你的职场命运。每一个选择都有代价，每一个代价都是成长。**
