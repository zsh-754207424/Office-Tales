# B3-1：独行侠

---

## 结局条件

| 项目 | 内容 |
|------|------|
| 结局编号 | B3-1 |
| 标题 | 独行侠 |
| 解锁条件 | 被孤立，不站队，无人帮 |
| 稀有度 | 5% |

---

## 场景：独行侠

```
你选择了不站队。
代价是被孤立。

同事们中午吃饭不再叫你了。
群里的消息，你总是最后一个看到的。

你坐在工位上，戴着耳机，假装很忙。

你知道发生了什么——
因为你没有选择任何一边。
所以两边都不把你当自己人。

你变成了"那谁"——
大家记不住你的名字，
但都知道你是"那个不站队的人"。

独行侠。
一个孤独的存在。
```

---

## 内心独白

```
我保持了自己的原则。
但我也付出了代价。

我在这里活下来了，但不快乐。

原则……是奢侈品。
我有资格拥有原则吗？
```

---

## 代价与获得

| 代价 | 获得 |
|------|------|
| 保持了自己的原则 | 原则 |
| 付出了代价——被孤立 | 清白 |
| 怀疑这一切是否值得 | 不欠任何人 |

---

## 角色对话

```
陈叔：
"你是个有原则的人。
但原则有时候是奢侈品。"

李总监：
"你要有自己的想法。
但也要学会融入团队。"
```

---

## 终场画面

```
你一个人坐在工位上。窗外是夕阳。

你收拾好东西，一个人下班。

没有人注意到你走了。

这就是独行侠的生活。
孤独，但不亏欠任何人。
```

---

> **独行侠 — 你的选择，决定了你的职场命运。每一个选择都有代价，每一个代价都是成长。**
