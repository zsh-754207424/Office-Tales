# B3-2：透明人

| 项目 | 内容 |
|------|------|

| 稀有度 | 5% |

---

## 场景：透明人

```
你完成工作。但没有人注意到你。

你在公司里就像一个"背景板"——
你在那里，但没有人看你。

开会时你从不发言。
提出方案时你从不主动。
讨论时你从不表达意见。

你是会议室里最安全的那个人。
也是最没有存在感的那个人。

有一天，前台打电话到你们部门：

"你好，请问小林在吗？"

你接起电话："我就是。"

前台愣了一下："哦……不好意思，我记不清你的声音。"

你挂了电话。你连前台都不记得你。

你是透明人。
存在，但不被看见。
```

---

## 内心独白

```
我选择了不引人注目。
但我也失去了存在感。

我存在。但没有"活着"。

这样活着有什么意义？

我不知道。
但我知道一件事——
我不想继续这样了。
```

---

## 代价与获得

| 代价 | 获得 |
|------|------|
| 你选择了不引人注目 | 安全 |
| 失去了存在感 | 无压力 |
| 开始思考这样活着有什么意义 | 不被针对 |

---

## 角色对话

```
李总监：
"你的工作还可以。
但你要有自己的想法。"

张姐：
"你太安静了。
在这里，安静是会被忽略的。"
```

---

## 终场画面

```
你收拾好东西，走出公司。

门口的保安看到你，愣了一下——
他好像也不记得你。

你苦笑。

明天，你还会来。后天，你也会来。
但有没有人知道你在这里？

也许……是时候做出改变了。
```
