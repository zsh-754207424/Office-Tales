---

## 场景：独狼战士

```
你选择了王副总的路线。

最初，你只是想活下去。
后来，你发现这条路比你想象的更"高效"。

王副总教你——
"关系比能力重要。站队比做事重要。"

你开始学会看人。知道谁有用，谁没用。

你升职了。

但有一天，你在电梯里看到一个新人——
他提着公文包，眼神里有你熟悉的东西。

那是一种"想做正确的事"的光芒。

你看着他。就像看着三个月前的自己。

电梯门开了。他走出去。你没有。
```

---

## 内心独白

```
我成功了。但我也变了。

我成为了当初我最讨厌的人。
我用同样的方式对待别人。

成功让我失去了初心。
但初心……能让我活下来吗？

我不知道。
但我知道，我已经没有回头路了。
```

---

## 代价与获得

| 代价 | 获得 |
|------|------|
| 成为了当初最讨厌的人 | 快速晋升 |
| 用同样的方式对待别人 | 派系庇护 |
| 成功让你失去了初心 | 职场生存能力 |

---

## 角色对话

```
王副总：
"你学得很快。在这里，这就是生存法则。"

陈叔：
"你……成了我。
我当初也是这样。"
```

---

## 终场画面

```
你站在办公室窗前，看着楼下的新人走进大楼。
你想起三个月前的自己。

然后你转身，继续开会。

没时间怀念了。
这是你选择的路。
```

---

> **独狼战士 — 你的选择，决定了你的职场命运。每一个选择都有代价，每一个代价都是成长。**
