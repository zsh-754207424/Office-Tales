# A2-1：工具人

---

## 结局条件

| 项目 | 内容 |
|------|------|
| 结局编号 | A2-1 |
| 标题 | 工具人 |
| 解锁条件 | 中立派系，李总监信任≥60，能力被利用但不被信任 |
| 稀有度 | 5% |

---

## 场景：工具人

```
你没有被裁。但也没有晋升。

你在公司里找到了一个独特的位置：
你有能力，但你不属于任何派系。

李总监交给你一些重要的工作。
但你注意到——真正核心的决策，他从来不让你参与。

有一天，你在茶水间听到两个同事的对话：

"小林那个人挺能干的。"
"是啊，但他是李总监的人吗？"
"不知道。他好像……谁的人都不是。"

你端着咖啡杯站在门外。
不知道该进去还是该转身。
```

---

## 内心独白

```
我是一个"好用的人"，但不是"自己人"。

我的能力是我的护身符，也是我的枷锁。
我可能永远无法进入核心圈子。
但我的能力让我不可或缺。

这……是一种平衡吗？
也许这就是我最好的位置了。
```

---

## 代价与获得

| 代价 | 获得 |
|------|------|
| 可能永远无法进入核心圈子 | 稳定 |
| 能力是你的护身符也是枷锁 | 不可替代性 |
| 被利用但不被信任 | 技能成长 |

---

## 角色对话

```
李总监：
"你的能力很强。
但在这里，光有能力是不够的。"

王浩（离职前）：
"你知道吗？你和我一样。
我们都只是……工具。"
```

---

## 终场画面

```
你回到工位，打开电脑，继续工作。

做一个"好用的人"，也许没什么不好。
至少……你还有价值。

至少你还在这里。
```

---

> **工具人 — 你的选择，决定了你的职场命运。每一个选择都有代价，每一个代价都是成长。**
