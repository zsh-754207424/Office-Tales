# A2-2：专业路线

---

## 结局条件

| 项目 | 内容 |
|------|------|
| 结局编号 | A2-2 |
| 标题 | 专业路线 |
| 解锁条件 | 中立派系，专注技术，不参与政治 |
| 稀有度 | 4% |

---

## 场景：专业路线

```
你选择了另一条路。

部门的派系斗争如火如荼。
你看着两边的人在明争暗斗。
而你戴着耳机，专心做自己的事。

王副总来找过你一次。
你礼貌地拒绝了。

李总监也试探过你一次。
你说："李总，我想专心做技术。"

你成为了公司的技术专家。
同事们有问题会来找你。
但你从不参与他们的"讨论"。

这不是懦弱。这是选择。
```

---

## 内心独白

```
我放弃了晋升的机会。
但我获得了专业上的尊重。

陈叔说，不是每个人都要站在最高处。
以前我不理解。
现在……我明白了。

我找到了自己的位置。
这就是我的路。
```

---

## 代价与获得

| 代价 | 获得 |
|------|------|
| 你放弃了晋升的机会 | 专业尊重 |
| 不参与权力游戏 | 心理平静 |
| 收入增长有限 | 技术权威 |

---

## 角色对话

```
李总监：
"你是我见过最专注的人。
这条路也很好。"

陈叔：
"你比我聪明。
我选择了站队，你选择了做事。"
```

---

## 终场画面

```
你的桌上放着一本技术杂志。
上面有你的文章。

你的名字旁边写着"技术专家"四个字。

你合上杂志，打开电脑。
又是新的一天。

有新的问题要解决。
有新的东西要学。

这就够了。
```

---

> **专业路线 — 你的选择，决定了你的职场命运。每一个选择都有代价，每一个代价都是成长。**
