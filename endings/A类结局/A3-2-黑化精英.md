## 场景：黑化精英

```
你不仅成功了。你还学会了这一套。

现在，你是公司里人人敬畏的角色。
你知道每个人的把柄。
你知道每件事的"规则"。

你坐在办公室里，面前是一个新人——他犯了错。
他看起来很害怕。

你看着他，想起了以前的自己。

然后你开口了：
"你知道你错在哪里吗？
你以为这是学校？做错了就能重来？
这里是职场。犯错，是要付出代价的。"

你的语气很冷。连你自己都觉得冷。

但这就是规则。
你只是……在教他规则。
```

---

## 内心独白

```
我成为了下一个王副总。
我失去了所有真正的朋友。
我变成了权力的奴隶。

但……这就是我选择的路。
我会走到底。

只是偶尔在深夜，
我会想起小林说的那句话：
"前辈，保持自己……"
```

| 变成了权力的奴隶 | 完全掌控 |
| 永远不会满足 | 没有人敢轻视你 |
