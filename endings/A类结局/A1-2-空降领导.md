## 场景：空降领导

```
你晋升了。成为公司最年轻的中层。

晋升公告贴在布告栏里，你的照片和简介在上面。
路过的同事看到你，露出复杂的表情——祝贺、嫉妒、疏远。

你的新办公室在八楼。从窗户看出去，能看到公司停车场。

你的下属来汇报工作——他们有的比你大，有的比你老。
他们叫你"林经理"，语气里有些生疏。

你坐在办公桌前，看着桌上新换的名牌。

你感觉自己比以前更孤独了。
```

---

## 内心独白

```
我成功了。我晋升了。
但我……并不快乐。

曾经和我一起吃饭的同事，现在看到我都会绕道。
他们叫我"林经理"，而不是"小林"。

这就是我要的成功吗？
为什么我感觉这么……空？

孤独是晋升的代价。
站在高处，就注定要忍受冷风。
```

---

## 代价与获得

| 代价 | 获得 |
|------|------|
| 孤独是晋升的代价 | 职位 |
| 必须做出一些艰难的决定 | 权力 |
| 开始怀疑这一切是否值得 | 薪资 |
| —— | 成就感 |

---

## 角色对话

```
秦总：
"你坐到这个位置，有什么感觉？
权力是孤独的，你准备好了吗？"

张姐（最后一次对话）：
"你变了，小林。
但也许……这是你必须变的。"
```

---

## 终场画面

```
你关上办公室的门。
窗外是城市的天际线，阳光照进来。

你深吸一口气。

这是你的选择。这是你的路。

但一个人在办公室里的感觉……真的很冷。
```

---

> **空降领导 — 你的选择，决定了你的职场命运。每一个选择都有代价，每一个代价都是成长。**
