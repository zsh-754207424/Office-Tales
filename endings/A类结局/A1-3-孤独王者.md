## 场景：孤独王者

```
庆功宴。你升职后的第一个周末。

你在KTV订了一个大包间，请了部门所有人。

但到了晚上8点，只有稀稀拉拉几个人来了。

张姐没来。小林没来——他已经走了。
那几个曾经和你一起吃饭的"朋友"也没来。

来的都是你的下属——他们不是来庆祝的，是来应酬的。

你坐在包间里，看着屏幕上滚动的歌词。

你突然觉得自己很可笑。

你买单的时候，前台说："先生，你们一共消费了2860元。"

你刷卡。走出KTV。风吹过来，你打了个寒颤。
```

---

## 内心独白

```
我赢了。但代价是什么？

那些曾经帮助过我的人，我没有回报。
那些曾经信任过我的人，我让他们失望了。

我成功了，但不快乐。
我得到了我想要的一切，但失去了更重要的东西。

这一切……值得吗？
```

---

## 代价与获得

| 代价 | 获得 |
|------|------|
| 你成功了，但不快乐 | 完全的胜利 |
| 得到了想要的一切，但失去了更重要的 | 职业巅峰 |
| 开始思考这一切是否值得 | 权力 |
| —— | 无人敢轻视 |

---

## 角色对话

```
秦总：

陈叔（梦中浮现）：

```

## 终场画面

```
你一个人坐在办公室里，看着窗外的夜景。
手机里没有一条来自"朋友"的消息。

只有工作群在响。

你赢了。
但你赢了什么？
```
