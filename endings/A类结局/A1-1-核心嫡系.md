## 场景：核心嫡系

```
三年后的某一天，你坐在李总监的办公室里。

不——准确地说，你坐在他办公室的对面。
你的办公桌搬到了这里——李总监主动提出的。

桌上的名牌写着你的名字和职位：产品部副总监。
窗外是熟悉的天际线。阳光透过玻璃照进来。

门推开，李总监走进来，手里拿着一份文件。

"小林，"他把文件放在你面前，"这个项目，你来做。"

你接过来看了一眼——公司今年最重要的战略项目。

他点点头，在你对面坐下。

"我下周出差。部门你盯着。"

他喝了一口茶，看着你。

那个眼神和以前不一样了。不再是审视。不再有怀疑。
是信任。

"这个部门……交给你了。"
```

---

## 内心独白

```
三年了。

三年前，我站在楼下，紧张得手心出汗。
三年后，我坐在副总监的位置上。

李总监说，这条路没有回头路。
他说得对。

我把所有时间都给了工作。
和朋友的关系淡了。
我开始理解他当年的选择。

但我也得到了什么？
尊重。地位。能力。成长。
还有一些真正的朋友——陈叔、张姐……

也许，这就是代价。
```

---

## 代价与获得

| 代价 | 获得 |
|------|------|
| 你把所有时间都给了工作 | 尊重 |
| 和朋友的关系淡了 | 地位 |
| 你开始理解李总监当年的选择 | 能力 |
| 头发白了几根 | 成长 |
| 忘记了上一次陪家人 | 真正的朋友 |

## 角色对话

```
秦总：
"小李跟我提过你，说你有潜力。
但你要想清楚，这条路没有回头路。"

李总监：
"三年后，我希望你能接我的班。
你能做到吗？"
```

## 终场画面

```
你站在办公室窗前，看着远处的天际线。
和第一天一样，阳光照在玻璃幕墙上。

但和第一天不一样的是——
你现在知道你是谁了。

你知道代价。
你也知道选择。
但你更知道——你选择的，就是你想要的。
```

---

> **核心嫡系 — 你的选择，决定了你的职场命运。每一个选择都有代价，每一个代价都是成长。**
