# 03-因果链条

## 核心因果原则

```
【因果链条基础】
每个事件必须是前一个事件的结果
巧合只能作为催化剂，不能作为解决方案
人物选择驱动情节发展
```

---

## 关键因果节点

### 因果节点1：午餐邀请

```
【因】
张姐邀请你午餐

【选项】
A. 接受
B. 拒绝

【果】
A. 张姐好感+15 → 她开始关注你 → 后续获得更多提点和情报
B. 张姐好感-10 → 她判断你"不好接近" → 后续关系冷淡

【因果链延伸】
无论A还是B，都会影响第3章站队时的选择
→ 不同的选择导致不同的盟友关系
→ 不同的盟友关系影响裁员时的处境
```

### 因果节点2：抢功事件

```
【因】
王浩把你的方案给领导看
领导表扬了他

【选项】
A. 当面质问
B. 沉默接受
C. 私下找领导
D. 收集证据

【果】
A. 公开对立 → 王浩敌意+30 → 后续持续对抗
B. 被动积累 → 王浩敌意+10 → 他觉得你好欺负
C. 私下博弈 → 王浩被批评 → 敌意+50，彻底撕破脸
D. 保留筹码 → 隐藏路线 → 第4章可能有爆发机会

【因果链延伸】
无论哪个选项，都无法回到"什么都没发生"
→ 和王浩的关系已经不可挽回地改变
→ 故事只能向前
```

### 因果节点3：派系站队

```
【因】
第3章，你需要选择立场

【选项】
A. 支持李总监
B. 中立
C. 疏远/倒向王副总

【果】
A. 李总监好感+20 → 进入嫡系 → 裁员时可能被保护
B. 无变化 → 两边都当你是外人 → 裁员时无人帮
C. 李总监信任归零 → 王副总可能拉拢 → 但价值观可能改变

【因果链延伸】
站队选择直接影响裁员结果
→ A类选项在裁员名单判定时有优势
→ C类选项如果被王副总抛弃，会很惨
```

### 因果节点4：王副总拉拢

```
【因】
王副总私下找你谈话

【选项】
A. 接受
B. 拒绝
C. 不表态

【果】
A. 李总监信任归零 → 加入王副总派系 → 价值观可能改变
B. 王副总成为敌人 → 后续可能被针对 → 但保持了立场
C. 两边不信任 → 成为"墙头草" → 两边都可能抛弃你

【因果链延伸】
这是价值观的关键选择
→ 选择A可能导致A3类结局（独狼/黑化）
→ 选择B可能获得李总监信任 → A1类结局
→ 选择C可能导致B类或C类结局
```

## 反例检查

### ❌ 避免的因果错误

```
【错误1：巧合解决】
错误：主角在裁员前"刚好"遇到秦总，秦总说"你不会被裁"
正确：主角的选择和行为决定了裁员结果

【错误2：凭空出现】
错误：第4章突然出现一个"神秘人物"帮助主角
正确：这个人物应该在第1-2章就已经铺垫

【错误3：选择无影响】
错误：主角做了选择，但没有任何后续影响
正确：每个选择都应该影响后续情节

【错误4：情感跳跃】
错误：主角被裁员后直接跳到"我很高兴离开"
正确：需要设计情感过渡
```

---

## 一句话总结

> **因果链条的核心：每个事件都有前因，每个选择都有后果，让故事成为一个有机整体**
