# 06-情感弧线设计

## 情感设计基础

### 核心情感模型

```
好奇 → 投入 → 担忧 → 紧张 → 悬念 → 惊讶 → 满足
  ↓       ↓       ↓       ↓       ↓       ↓       ↓
入职    任务    站队    裁员    结局    揭示    反思
期待    焦虑    纠结    恐惧    好奇    意外    释然
```

---

## 每章情感设计

### 第一章：入职适应

```
情感主线：好奇与期待

【开场】
好奇：这是一个新的开始
期待：我会遇到什么人？会发生什么？

【中期】
紧张：面对领导的第一次汇报
焦虑：不知道自己做得好不好

【结尾】
愤怒/憋屈：抢功事件
失望：原来职场是这样的

情感强度曲线：
      😊😊😊😊😊
     /
    /         😟
   /         /
  /    🤔   /
 /    /     /
───────────────→ 时间
入职   适应   汇报   抢功
好奇   期待   紧张   愤怒
```

### 第二章：首个任务

```
情感主线：投入与焦虑

【开场】
压力：任务来了，我能做好吗？
焦虑：遇到困难，不知道怎么办

【中期】
坚持：加班加点，努力完成
担心：会不会出问题？

【结尾】
憋屈：被抢功
愤怒：为什么是我？
无奈：我要怎么办？

情感强度曲线：
      😊
     /
    /         😰
   /    😟   /  \
  /    /    /    \
 /    /    /      \
───────────────────→ 时间
任务   困难   完成   抢功
投入   焦虑   紧张   憋屈
```

### 第三章：团队协作

```
情感主线：担忧与纠结

【开场】
希望：季度评估，可能被表扬
紧张：要在大会上发言

【中期】
纠结：要站队吗？站哪边？
担忧：站错了怎么办？

【结尾】
确认：站队结果确定
紧张：派系斗争开始
担忧：裁员越来越近

情感强度曲线：
      😊
     / \
    /   \         😟
   /     \       /  \
  /       \     /    \
 /         \   /      \
───────────────────────→ 时间
评估   发言   站队   暗流
希望   紧张   纠结   担忧
```

### 第四章：利益冲突

```
情感主线：紧张与恐惧

【开场】
紧张：裁员要来了
恐惧：会不会有我？

【中期】
焦虑：各种暗示和情报
绝望：情况不太妙
挣扎：要不要做点什么？

【结尾】
震惊：名单公布
释然/绝望：我的名字在/不在
无奈：命运已定

情感强度曲线：
      😰
     /  \
    /    \         😱
   /      \       /  \
  /        \     /    \
 /          \   /      \
────────────────────────→ 时间
预警   拉拢   对抗   名单
紧张   焦虑   挣扎   震惊
```

### 第五章：结局

```
情感主线：悬念与释然

【开场】
紧张：结果是什么？
好奇：我的选择对了吗？

【中期】
惊讶：原来是这种结局
意外：代价是这样的？

【结尾】
反思：我想成为什么样的人？
释然：这就是我的选择
接受：这就是我的人生

情感强度曲线：
      😮
     /  \
    /    \         🤔
   /      \       /  \
  /        \     /    \
 /          \   /      \
────────────────────────→ 时间
揭晓   对话   对照   反思
紧张   惊讶   共鸣   释然
```

## 憋屈→释放设计

### 核心设计原则

```
【憋屈是必须的】
- 让玩家在前2章"憋着"
- 1-2个选项都是憋屈的
- 等玩家"忍无可忍"时，给一个逆袭机会

【释放是爽的】
- 反差产生的爽感最强
- 憋屈越久，释放越爽
- 但释放也要有代价
```

### 憋屈→释放节奏

```
第1章：
憋屈：被忽视，被当空气
释放：入职第一天结束，张姐说"明天一起吃饭"

第2章：
憋屈：被抢功，努力白费
释放：如果选择D，第4章爆发，让王浩付出代价

第3章：
憋屈：站队困难，两难选择
释放：盟友确认，你不是一个人

第4章：
憋屈：裁员危机，恐惧不安
释放：名单公布，你留下了/你赢了

第5章：
憋屈：揭示代价，失去的东西
释放：选择"我想成为什么样的人"，你掌握了主动权
```

## 情感高潮设计

### 第一高潮：抢功事件（第2章）

```
【为什么是第一高潮】
- 第一个重大冲突
- 憋屈感最强
- 选择后的后果最直接

【情感强度】
愤怒：★★★★☆
憋屈：★★★★★
无奈：★★★★☆

【玩家感受】
"凭什么！"
"我要让他付出代价！"
"这个职场太黑暗了！"
```

### 第二高潮：裁员名单（第4章）

```
【为什么是第二高潮】
- 命运的终极转折
- 紧张感最强
- 结果不可改变

【情感强度】
紧张：★★★★★
恐惧：★★★★☆
期待：★★★☆☆

【玩家感受】
"会不会有我？"
"我的选择对了吗？"
"接下来会发生什么？"
```

### 第三高潮：代价揭示（第5章）

```
【为什么是第三高潮】
- 最终的情感释放
- 反思感最强
- 意义的终极呈现

【情感强度】
惊讶：★★★★☆
反思：★★★★★
释然：★★★★☆

【玩家感受】
"原来是这样……"
"我得到了，也失去了"
"这就是我的选择"
```

## 情感曲线检查表

```
【每个场景都要问】
1. 这个场景的主要情感是什么？
2. 情感强度够不够？
3. 和前后场景的情感是否衔接？
4. 有没有给玩家情绪释放的机会？
5. 憋屈和爽的比例是否合适？

【情感弧线检查】
1. 整体情感是否向上？
2. 是否有起伏？
3. 高潮是否足够高？
4. 低谷是否足够低？
5. 结局是否让人满足？
```

---

## 一句话总结

> **情感设计的核心：让玩家经历完整的情感旅程，从好奇到愤怒，从恐惧到释然，从选择到接受**
