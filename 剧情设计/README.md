# 剧情设计总览

## 项目概述

职场剧情互动小游戏，通过选择走向不同的职业命运。

## 核心框架

### LOCK系统
- **L（主角）**：职场新人（你）
- **O（目标）**：在公司生存并获得晋升
- **C（冲突）**：李总监派 vs 王副总派 + 王浩的竞争
- **K（结局）**：24种不同的职业命运

### 三幕结构
```
第一幕（1-2章）：入职适应 → 站稳脚跟
第二幕（3-4章）：成长与抉择 → 派系站队
第三幕（5章）：结局收束 → 命运揭示
```

### 关键转折
- **第一扇门**：第2章抢功事件（关系转折）
- **第二扇门**：第4章裁员名单（命运转折）

## 数值系统

### 信任/好感度
- 李总监信任：0-100
- 王浩敌意：0-100
- 陈叔好感：0-100
- 张姐好感：0-100
- 小刘HR好感：0-100

### 结局判定
- A类：信任≥60 + 派系正确 + 关键选择正确
- B类：信任40-59 + 中立或轻度支持
- C类：信任<40 或 站错队
