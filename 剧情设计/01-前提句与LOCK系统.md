# 01-前提句与LOCK系统

## 前提句

> **一个职场小白进入充满派系斗争的公司，在"能力"与"关系"的夹缝中求存，通过一系列选择，最终走向不同的职业命运。**

---

## LOCK系统详解

### L - Lead（主角）

**职场新人（你）**
- 入职时间：第1天
- 背景：普通院校毕业，无背景无资源
- 性格：由玩家选择决定
- 起点能力：基础工作能力，无人脉无经验

**主角特征设计**
```
【可塑性强】
- 没有固定人设，由选择塑造
- 玩家的每一个决定定义"你"是谁
- 你的选择决定你成为什么样的人

【代入感高】
- 从零开始，和玩家一起成长
- 面对的困难是每个职场人都会遇到的
- 选择的困境是真实存在的
```

### O - Objective（目标）

**短期目标（每章）**
```
第1章：适应环境，获得初步认可
第2章：完成首个任务，证明能力
第3章：融入团队，获得支持
第4章：度过裁员危机
第5章：明确职业方向
```

**中期目标（贯穿全剧）**
```
- 在公司站稳脚跟
- 获得晋升或找到合适的位置
- 建立有价值的人际关系
- 找到自己的职场价值观
```

**终极目标（由玩家定义）**
```
- 晋升到高位？
- 获得稳定生活？
- 实现自我价值？
- 保持初心不被改变？
```

### C - Conflict（冲突）

**三层冲突结构**

```
【表层冲突：个人层面】
- 同期入职竞争（王浩）
- 工作能力证明
- 领导第一印象

【中层冲突：人际层面】
- 融入团队
- 建立盟友
- 获得信任

【深层冲突：价值观层面】
- 能力 vs 关系
- 个人 vs 团队
- 原则 vs 生存
```

**核心冲突矩阵**

| 冲突方 | 目标 | 立场 |
|--------|------|------|
| 王浩 | 压制你往上爬 | 对立 |
| 李总监 | 培养可用之人 | 观察你 |
| 王副总 | 拉拢或打压你 | 试探你 |
| 张姐 | 自保 + 观察新人 | 中立友善 |
| 陈叔 | 等待值得教的人 | 观望 |

### K - Knockout（结局）

**24种结局分布**

```
A类（晋升）- 9种
├─ A1-1：核心嫡系
├─ A1-2：空降领导
├─ A1-3：孤独王者
├─ A2-1：工具人
├─ A2-2：专业路线
├─ A2-3：外派高升
├─ A3-1：独狼战士
└─ A3-2：黑化精英

B类（稳定）- 7种
├─ B1-1：稳健派
├─ B1-2：老黄牛
├─ B2-1：被利用的棋子
├─ B2-2：夹缝生存
├─ B3-1：独行侠
└─ B3-2：透明人

C类（离职）- 8种
├─ C1-1：优雅转身
├─ C1-2：创业新星
├─ C1-3：回归家庭
├─ C2-1：派系牺牲品
├─ C2-2：被迫离开
├─ C2-3：暗箱操作
├─ C3-1：自然淘汰
└─ C3-2：职业倦怠
```

## 故事主题

### 核心主题
**"在职场，你选择成为什么样的人？"**

### 次级主题
```
1. 能力与关系，哪个更重要？
2. 坚持原则 vs 适者生存
3. 成功需要付出什么代价？
4. 什么是真正的职场成长？
5. 你愿意为了成功放弃什么？
```

### 主题层次

```
【表面层】
职场生存指南

【中间层】
人性的选择与代价

【深层】
关于"成功"的定义
关于"自我"的坚持
关于"代价"的认知
```

---

## 玩家体验设计

### 代入感设计
```
【从零开始】
- 玩家和主角一起成长
- 没有先入为主的设定
- 每一个选择都有重量

【真实困境】
- 每个选择都是真实职场会遇到的两难
- 没有绝对正确的答案
- 后果即时可见

【因果闭环】
- 你的选择塑造你的角色
- 你的选择决定你的结局
- 结局揭示你一路走来的选择
```

### 决策质量设计
```
【信息不对等】
- 玩家无法知道所有后果
- 选择时只能凭直觉和判断
- 体现了真实决策的困难

【代价可见】
- 短期利益 vs 长期影响
- 明确展示每个选择的价值
- 让玩家理解选择的意义

【无后悔药】
- 关键选择无法反悔
- 体现了决策的不可逆性
- 增加选择的重量
```

---

## 一句话总结

> **LOCK系统的核心：让玩家通过选择定义主角，通过主角的经历反思自己**
