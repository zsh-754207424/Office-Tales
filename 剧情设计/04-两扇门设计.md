# 04-两扇门设计

## 两扇门的概念

```
【第一扇门】
位置：第2章结尾
性质：关系转折门
功能：故事方向开始分化，无法回头

【第二扇门】
位置：第4章结尾
性质：命运转折门
功能：主角命运确定，故事进入揭示阶段
```

---

## 第一扇门：抢功事件

### 门的位置与时机

```
章节：第2章 - 首个任务
时机：完成任务后，王浩汇报的那一刻
触发：自动触发，无法跳过
```

### 门的场景设计

```
【场景】
你完成了第一个任务，做了一份市场分析报告
你正准备向李总监汇报
王浩走过来："你的报告我看了，帮你优化了一下，已经发给李总监了"
李总监回复："小王这个报告做得不错，有洞察"

你看到王浩在群里被表扬
你的名字没有被提到

周围同事的反应：
- 张姐看了你一眼，什么都没说
- 王浩笑着对你说："谢了啊"
- 其他同事没有反应
```

### 门的三个层次

```
【第一层：表面冲突】
王浩抢了你的功劳
→ 你要不要当场发作？

【第二层：关系转折】
无论你怎么选，和王浩的关系都不可挽回地改变
→ 他知道你会记仇
→ 你知道他不是好人
→ 你们无法回到"普通同事"关系

【第三层：性格揭示】
你的选择揭示了你的性格
→ 质问 → 你是有底线的人
→ 沉默 → 你是能忍的人
→ 私下 → 你是聪明的人
→ 收集证据 → 你是有心机的人
```

### 门的选择与后果

| 选项 | 即时后果 | 后续影响 |
|------|----------|----------|
| **A.当面质问** | 王浩当场否认，场面尴尬 | 王浩敌意+30，公开对立，后续持续对抗 |
| **B.沉默接受** | 王浩笑着说"谢了"，你咽下这口气 | 王浩敌意+10，觉得你好欺负 |
| **C.私下找领导** | 李总监了解了情况，批评王浩 | 王浩敌意+50，彻底撕破脸，但领导知道真相 |
| **D.收集证据** | 默默记下这件事 | 隐藏路线，保留筹码，第4章可能有爆发机会 |

### 门的不可逆性

```
【为什么不可逆？】

1. 王浩知道你在意这件事
   → 他会警惕你
   → 他会继续针对你

2. 你知道了王浩是什么人
   → 你无法再信任他
   → 你无法假装什么都没发生

3. 同事们看到了这一幕
   → 他们会站队或观望
   → 你的处理方式影响他们对你的看法
```

### 门后的事件链

```
【选A后】
→ 王浩开始公开针对你
→ 同事们知道你们关系不好
→ 第3章可能出现更多摩擦
→ 第4章裁员时，王浩可能踩你

【选B后】
→ 王浩觉得你好欺负
→ 可能会继续占你便宜
→ 但也给你时间积累实力
→ 第3章你可能找到机会反击

【选C后】
→ 王浩恨你入骨
→ 但领导知道真相
→ 你可能获得领导信任
→ 第4章可能是生死对决

【选D后】
→ 你的选择暂时不产生变化
→ 但你有了筹码
→ 第4章王浩可能再次犯事
→ 你的证据可能成为关键
```

## 第二扇门：裁员名单

### 门的位置与时机

```
章节：第4章 - 利益冲突
时机：完成所有关键选择后
触发：自动触发，无法跳过
前置条件：所有选择节点已完成
```

### 门的场景设计

```
【场景】
今天早上，HR群里发了一条消息：
"请各部门负责人10点到会议室"

气氛突然紧张起来

张姐从会议室出来，脸色不太好
她经过你身边，小声说："名单出了"

王浩从会议室出来，脸上有压抑不住的笑容
他看了你一眼，没有说话

李总监走回办公室
他经过你身边，停了一下
看了你一眼，叹了口气
然后继续往前走

小刘HR从会议室出来
她看到你，犹豫了一下
然后轻轻摇了摇头
```

### 门的三个层次

```
【第一层：命运揭晓】
裁员名单公布
→ 你看到自己的名字在/不在上面
→ 你的命运已经确定

【第二层：原因揭示】
裁员名单不是随机的
→ 是你全程选择的结果
→ 是派系博弈的结果
→ 是能力+关系+立场的综合

【第三层：不可逆转】
无论结果如何，故事只能向前
→ 被裁：需要处理离职
→ 没被裁：需要面对后续
→ 故事进入最终揭示阶段
```

### 门后的可能结果

| 结果 | 条件 | 后续走向 |
|------|------|----------|
| **名单上有你** | 信任<40 或 站错队 | C类结局（离职） |
| **名单上有王浩** | 你获得信任+正确选择 | A类结局（晋升） |
| **名单有你和王浩** | 中立或平局 | B类结局（稳定） |

### 门的不可逆性

```
【为什么不可逆？】

1. 名单已经确定
   → 你无法改变结果
   → 你只能接受或应对

2. 派系格局已定
   → 李总监派赢了/输了
   → 你的立场已经明确
   → 没有中间路线

3. 故事进入最终阶段
   → 剩下的只是揭示和对话
   → 没有新的选择节点
   → 只有结局的选择
```

## 两扇门的对比

| 维度 | 第一扇门 | 第二扇门 |
|------|----------|----------|
| **位置** | 第2章结尾 | 第4章结尾 |
| **性质** | 关系转折 | 命运转折 |
| **影响** | 人际关系 | 职业命运 |
| **可逆性** | 完全不可逆 | 完全不可逆 |
| **选择数量** | 4个选项 | 无选择（结果已定） |
| **后续影响** | 影响第3-4章 | 影响第5章结局 |
| **情感冲击** | 愤怒/憋屈 | 震惊/释然/绝望 |

## 门的设计原则

### 1. 提前铺垫

```
【第一扇门铺垫】
- 第1章提到王浩"表面热情"
- 张姐暗示"有些人喜欢抢功"
- 李总监说"我只看结果，不听借口"

【第二扇门铺垫】
- 第3章提到"公司最近有变动"
- 陈叔说"裁员不是裁能力差的"
- 小刘HR说"你做好心理准备"
```

### 2. 必然触发

```
【第一扇门】
- 只要你完成任务，就一定会被抢功
- 这是王浩的人设，不是巧合
- 目的是制造核心冲突

【第二扇门】
- 裁员是剧情必然
- 是公司经营问题，不是针对你
- 目的是制造终极冲突
```

### 3. 情感冲击

```
【第一扇门】
- 让玩家感到憋屈
- 让玩家想要反击
- 激发玩家的情绪

【第二扇门】
- 让玩家紧张
- 让玩家回顾选择
- 引出玩家的反思
```

### 4. 后续影响

```
【第一扇门】
- 改变王浩关系
- 影响第3章站队
- 影响第4章裁员立场

【第二扇门】
- 确定结局类型
- 影响最终对话
- 影响结局揭示
```

---

## 一句话总结

> **两扇门的灵魂：它们是不可回头的悬崖，让玩家的每一个选择都变得有重量**
