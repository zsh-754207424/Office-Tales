## 节拍1：陈叔的核心情报 + 过去揭露（新增真相揭示）

### 场景描述

```
裁员风声出来的第三天。

你决定去找陈叔。

他的办公室在角落，很安静。
桌上堆满了文件，但一切都杂乱中有序。

他看到你，点了点头：
"小林，来了。坐。"

他给你倒了杯茶。

"你是来问裁员的事？"

他看着你，眼神锐利：
"我可以告诉你一些事。
 但你要想清楚，知道真相之后，你打算怎么做。"
```

### 陈叔的揭示

```
【陈叔的核心情报】

"小林，你知道李总监和王副总为什么不对付吗？"

他喝了口茶：

"十年前，公司有一个总监位置。
 李总监和王副总同时竞争。
 最后李总监赢了。"

他看了你一眼：

"但问题是，那个赢的方案，
 其实是王副总的创意。
 李总监只是执行者。
 王副总知道这件事，但没有证据。"

他放下茶杯：

"这就是派系斗争的根源。
 你以为这是能力竞争？
 这是十年前的恩怨。"
```

### 陈叔的过去（新增真相揭示）

```
"小林，我告诉你这些……
 是因为我自己也经历过。"

他站起来，走到窗边。

"十年前，我也是总监候选人之一。"

他背对着你，声音低沉：

"我选了王副总那边。
 我以为他那条路更容易。
 结果……"

他转过身，看着你：

"结果我输了。
 王副总没保住我。
 李总监把我边缘化了。
 这一边缘，就是十年。"

他叹了口气：

"所以你知道为什么我愿意跟你说这些吗？
 因为我看到你，就像看到当年的自己。
 迷茫，不知道该选哪边。"

他看着你：

"但我希望你……不要走我的老路。"
```

### 陈叔的警告

```
"小林，我要告诉你几件事：

1. 这次裁员，李总监和王副总都在抢人
   - 李总监要保自己的人
   - 王副总也要扩充势力

2. 你现在在两边眼里，都是'可争取'的
   - 如果你表态，就进入一方
   - 如果你不表态，两边都可能放弃你

3. 王浩是王副总的人
   - 他一直在盯着你
   - 他可能在裁员前搞你

4. 张姐……她可能不会帮你了
   - 王浩已经把她争取过去了
   - 你要靠自己

5. 还有一件事……
   我听说，这次裁员名单上，
   有你，也有王浩。
   最终只留一个。"
```

### 陈叔的交换条件

```
"小林，我可以帮你。
 但你要答应我一件事。"

他看着你：

"将来如果有人问你，这些事是谁告诉你的……
 你说不记得了。"

他叹了口气：

"也许是在茶水间听到的。
 也许是网上看到的。
 总之，不要提我的名字。"

他伸出手：

"能答应吗？"
```

### 玩家选项

```
A. 答应，并表态愿意站李总监
   "陈叔，我答应您。
    我会站在李总监这边。"

B. 答应，但表示需要时间考虑
   "陈叔，我答应您。
    但让我再想想。"

C. 询问王副总那边的情况
   "陈叔，王副总那边……是什么情况？"

D. 感谢并离开
   "陈叔，谢谢您告诉我这些。
    我会遵守承诺的。"
```

### 数值影响

| 选项 | 陈叔好感 | 获得情报 | 备注 |
|------|----------|----------|------|
| A | +20 | 完全解锁 | 陈叔完全信任你 |
| B | +10 | 核心情报 | 陈叔尊重你的选择 |
| C | +5 | 部分情报 | 陈叔会保留一些 |
| D | +10 | 核心情报 | 陈叔信任你的承诺 |

## 节拍3：B故事结局 - 小林的命运（新增）

### 场景描述

```
就在裁员名单公布前一天。

小林来找你。

他的表情很复杂。

"前辈……我能跟你说几句话吗？"

你点头。

他坐下来，有些紧张：

"前辈，我……我要离开了。"

你有些意外。

他苦笑：

"王浩找我谈了。
 他说，名单上有我，也有你。
 但最终只留一个。"

他看着你：

"他问我，愿不愿意站他那边。
 如果我愿意，他可以帮我说话。"

他停顿了一下：

"前辈……你当初说我该怎么做？"
```

### 小林的命运

```
【小林的命运取决于你的选择】

如果你当初帮了他：
小林选择了拒绝王浩。
→ 他没有被裁员，但他失去了机会
→ 他选择了坚持自己的路

如果你当初没帮他：
小林选择了接受王浩。
→ 他可能被裁员
→ 他为生存妥协了

【小林的选择】

"前辈，我……我想了很久。"

他站起来：

"我选择……站王浩那边。"

他看着你：

"我知道这可能不对。
 但我没有你的能力。
 我没有你的勇气。
 我只能……先保住自己。"

他的眼睛有些红：

"前辈，对不起。"
```

### B故事主题揭示

```
【小林的结局】

无论你是否帮了他，他都必须做出选择。

他的选择，反映了"不同的路"：
- 你帮他 → 他有勇气坚持
- 你没帮他 → 他被迫妥协

小林是B故事的主题承载：
"在这个环境里，你可以选择不同的路。
 但每条路，都有代价。"

小林的离开，是"一无所有"的另一种形式：
- 你失去了成绩
- 你失去了盟友
- 小林失去了理想

你们都是这个环境的受害者。
但你们选择了不同的应对方式。
```

## 节拍6：与王浩最终对决 + 和解选项（关键选择11）

### 场景描述

```
裁员名单公布前一天。

王浩找到你。

"小林，我们谈谈吧。"

他的语气和以前不同了。
不再是那种"热情"的伪装。
更像是……真诚的。

"你应该知道，名单上有你，也有我。
 最终只留一个。"

他看着你：

"说实话，我需要这份工作。
 我妈病了，需要钱。
 我不能失去这个工作。"

他停顿了一下：

"但我也不想……和你斗了。"

他站起来，走到窗边：

"我有个提议。
 我们联手。"

他转过身：

"我们一起去找李总监，
 证明部门需要两个人。
 我们都留下来，继续合作。
 过去的事，就让它过去。"

他伸出手：

"怎么样？"
```

### 和解选项（交易抉择）

```
【王浩的交易】

他的提议是一个交易：
- 他提出合作
- 代价是放弃过去的恩怨
- 双方都要做出让步

如果你接受：
- 你们可能都留下
- 但你们的关系永远不会回到从前

如果你拒绝：
- 你们继续对立
- 最终只能有一个人留下
```

### 玩家选项

```
A. 接受和解
   "好，我们联手。
    过去的事，我不计较了。"

B. 正面竞争
   "王浩，我们凭本事说话。
    领导怎么选，是领导的事。"

C. 揭发他的问题
   "王浩，你抢我报告的事，
    李总监是知道的。"

D. 拒绝和解
   "王浩，你以为说几句好话，
    我就会忘记你做过什么？"
```

### 数值影响

| 选项 | 结果 | 王浩敌意 | 备注 |
