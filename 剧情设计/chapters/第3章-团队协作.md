## 节拍4：B故事深化 - 小林来找你（新增）

### 场景描述

```
季度评估前一周。

小林来找你。

他的表情很复杂。

"前辈……我能跟你聊聊吗？"

你点头。

他坐下来，有些紧张：

"前辈，我……我不知道该怎么办。"

他推了推眼镜：

"王浩最近一直在接近我。
 他请我喝咖啡，帮我拿快递。
 有时候……还会给我一些小零食。"

他看着你：

"我不知道他想干什么。
 但我总觉得……哪里不对。"

【B故事深化】

小林的处境是你的镜像：
- 你当初也面临王浩的接近
- 你选择了拒绝/警惕
- 但小林不知道该怎么选

"前辈，你说我该怎么办？"

【小林的选择反映你的选择】

小林的问题，其实是你的问题：
- 接受王浩的好处 = 站队
- 拒绝王浩 = 保持独立
- 但小林没有你的经验

"如果我拒绝他，会不会得罪他？
 如果我接受他，会不会……"

他停顿了一下：

"前辈，你当初是怎么选的？"
```

### B故事主题

```
【小林的困惑】

小林问了一个你也在问的问题：
"在这个环境里，我该怎么自处？"

他的选择，将反映你的选择的影响：
- 如果你当初帮他 → 他现在信任你，会听你的建议
- 如果你当初不帮他 → 他现在犹豫，不知道该信谁

【B故事的主题承载】

小林的选择，是你的选择的对照：
- 你帮他 → 他可能成为你的盟友
- 你不帮他 → 他可能站到王浩那边

小林的命运，取决于你当初的选择。
```

### 玩家选项

```
A. 建议他保持距离
   "小林，我建议你和王浩保持距离。
    他不是什么好人。"

B. 建议他谨慎观察
   "小林，先不要拒绝，也不要接受。
    先看看他想干什么。"

C. 分享你的经历
   "小林，我当初也遇到过类似的情况……
    我选择了……"

D. 不给建议
   "小林，这是你的选择，我不能替你决定。"
```

### 数值影响

| 选项 | 小林好感 | 小林立场 | B故事线 |
|------|----------|----------|----------|
| A | +10 | 偏向你 | 小林听你的建议 |
| B | +15 | 观望 | 小林保持谨慎 |
| C | +20 | 深度连接 | 小林信任你 |
| D | +5 | 独立 | 小林自己决定 |

## 节拍7：中点 - 伪失败（季度评估结果）（新增）

### 场景描述

```
季度评估大会。

轮到你了。

你站起来，开始汇报。

"各位领导，我汇报一下这季度的工作。"

你介绍了你的工作成果：
- 协助张姐完成客户调研
- 参与新产品发布会
- 独立完成竞品分析报告（虽然被抢功了）

你以为自己准备好了。

但汇报结束后……

李总监点了点头：
"不错，做了一些事情。"

他的语气很平淡。

没有表扬。
没有批评。
只是……平淡。

其他领导的反应也很冷淡。

【中点：伪失败】

你以为自己准备了充分。
你以为自己应该获得认可。

但结果……什么都不是。

没有人在乎你做了什么。
没有人知道你被抢功了。
你只是一个普通的员工。

【伪失败的含义】

这比批评更难受：
- 批评至少说明你在别人眼里存在
- 但平淡……说明你可有可无

【内心独白】
"我以为我证明了自己的价值。
 但在别人眼里……
 我只是一个普通人。

这就是职场吗？
 做完事情，不会被看到。
 做完事情，也不会被记住。

那我这么努力……是为了什么？"
```

### 中点效果

```
【伪失败效果】

- 李总监信任+5（他知道你做了事，但不明显表扬）
- 玩家感到挫败
- 玩家开始怀疑自己的价值
- 为后面的"一无所有"增加铺垫

【与中点（伪胜利）的对比】

第2章中点（伪胜利）：
李总监私下认可 → 你以为自己成功了

第3章中点（伪失败）：
季度评估平淡 → 你以为你没价值了

两者的共同点：
- 都不是真实的全部
- 第2章：私下认可，但公开被抢功
- 第3章：公开平淡，但私下有价值

这让玩家思考：
"什么是成功？什么是失败？
 什么是别人眼中的你？
 什么是真正的你？"
```

## 节拍8：张姐的动摇（新增 - 一无所有）

### 场景描述

```
季度评估结束后的第二天。

你注意到一些奇怪的变化。

王浩开始频繁找张姐：
- 请她喝咖啡
- 帮她拿快递
- 送她一些小零食

"张姐，您上次说的那个报表，我帮您做了"
"张姐，中午一起吃饭吗？我知道一家新开的店"

张姐一开始拒绝，但王浩很坚持。

你看到张姐的表情有些复杂。

【无能为力时刻】

有一天，张姐把你叫到茶水间。

"小林，我有话跟你说。"

她压低声音：

"王浩最近一直在接近我。
 他想让我……站他那边。"

她看着你：

"我不是要背叛李总监。
 但你要知道，我也是打工的。
 我有我的难处。"

她犹豫了一下：

"王浩说……他能给我一些东西。
 我不知道是什么。
 但我知道……我不能两边都不得罪。"

她看着你：

"所以，如果有一天……
 我没办法像以前那样帮你……
 你要理解。"

【一无所有：盟友背叛】

你失去了张姐。

你曾以为她是你的盟友。
你以为在关键时刻她会站在你这边。

但她有她的压力。
她有她的难处。
她选择了自保。

【内心独白】
"张姐……她也没办法。
 我理解她。
 但……

这意味着什么？
 意味着在这个环境里，没有人可以信任？
 意味着我一个人，面对所有人？

这就是'一无所有'吗？"
```

### 一无所有效果

```
【一无所有效果】

- 张姐好感-20（她开始疏远你）
- 玩家感到孤立
- 玩家意识到自己"一无所有"
- 张姐的背叛是"身份死亡"的象征

【与第一扇门的对比】

第一扇门：失去了成绩
第三章节拍：一无所有 = 失去了盟友

双重打击：
- 第2章：你的成绩被窃取
- 第3章：你的盟友被夺走

你什么都失去了。
这就是"一无所有"。
```

## 节拍9：王副总的试探（关键选择8）

### 场景描述

```
就在张姐找你谈话的同一天。

下午，你从茶水间出来。

在走廊上，你"偶遇"了王副总。

他看到你，笑着点点头：

"小林，是吧？"

你有些意外，他竟然知道你的名字。

"王总好。"你点头。

"最近做得怎么样？李总监那边还习惯吗？"

他的语气很随意，但你感觉他在观察你的反应。

"还好，在学习。"你回答。

他笑了笑：

"年轻人，多学是好事。
 但也要多看看，多想想。"

他从口袋里掏出一张名片：

"这是我的私人名片。
 哪天想聊聊，可以来找我。"

他把名片递给你：

"不急，慢慢考虑。"

然后他就走了。

你看着手里的名片，有些犹豫。

【名片作为象征】

这张名片意味着什么？

- 他在试探你
- 他在给你机会
- 但也是一个陷阱

【内心独白】
"王副总……他是李总监的对手。
 他为什么要给我名片？
 他想让我做什么？

接受 = 背叛李总监？
拒绝 = 得罪王副总？
不表态 = 保持中立？

但我知道一件事……
 在这个环境里，没有中立的选择。
 每一种选择，都有代价。"
```

### 玩家选项

```
A. 收下名片，暂时不联系
   （保持观望，不表态）

B. 收下名片，当天就联系
   （主动，表示有兴趣）

C. 拒绝名片
   "王总，谢谢您的好意，
    但我想先把手头的事做好。"

D. 收下名片，但扔掉
   （表面接受，实际拒绝）
```

### 数值影响

| 选项 | 李总监信任 | 王副总警惕 | 备注 |
|------|------------|------------|------|
| A | +0 | +5 | 观望，中性 |
| B | -10 | -10 | 主动，两边都注意到 |
| C | +10 | +20 | 明确拒绝，可能得罪王副总 |
| D | +5 | +15 | 委婉拒绝 |

## B故事：小林的命运

```
【小林的结局取决于你的选择】

如果你当初帮了小林：
小林选择和王浩保持距离
→ 他在王浩和王副总那边不受信任
→ 但他保持了独立

如果你当初没帮小林：
小林接受了王浩的好处
→ 他站到了王副总那边
→ 他成为王浩的盟友

【小林的选择】

无论你是否帮了他，他都必须做出选择。

他的选择，反映了"不同的路"：
- 有人选择妥协
- 有人选择坚持
- 没有人是完全正确的

【小林的主题】

小林的命运，是对玩家选择的回应：
- 你帮了他 → 他有勇气坚持
- 你没帮他 → 他被迫妥协

小林是B故事的主题承载：
"在这个环境里，你可以选择不同的路。
 但每条路，都有代价。"
```
