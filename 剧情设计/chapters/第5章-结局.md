## 节拍1：结局揭晓（命运判定）

### A类结局：晋升

```
【场景】
你没有被裁。
李总监把你叫到办公室。

"小林，坐。"

他看着你，表情比平时温和了一些。

"这次裁员，你没有被裁。
 说明你有价值。"

他站起来，走到窗边。

"但这只是开始。
 接下来的路，会更难走。"

他转过身，看着你：

"你知道为什么我选你吗？"

他走到你面前：

"不是因为你多聪明。
 是因为你……"

他停顿了一下：

"你让我想起年轻时的自己。
 有原则，但不死板。
 能忍，但也有底线。"

他拍了拍你的肩膀：

"好好干。
 三年后，我希望你能接我的班。"
```

### B类结局：稳定

```
【场景】
你没有被裁，但也没有晋升。

李总监把你叫到办公室。

"小林，坐。"

他看着你，表情平静。

"这次裁员，你留下来了。
 说明你还有价值。"

他顿了顿：

"但你要清楚，
 你还有很大的进步空间。"

他站起来，走到窗边。

"我不期望每个人都成为顶尖。
 但我希望每个人都能发挥自己的价值。"

他转过身：

"继续努力。
 这里需要你这样的人。"
```

### C类结局：离职

```
【场景】
你被裁了。

小刘HR找你办手续。

"小林，这是你的离职手续。
 有什么问题吗？"

她的语气很平静，但你听出了同情。

"我帮你争取了一个月的缓冲期。
 你可以慢慢找下家。"

她递给你一叠文件：

"这是你的离职证明、推荐信……
 有什么需要帮忙的，可以找我。"

你接过文件，心情复杂。
```

## 节拍2：B故事结局 - 小林的最终命运

### 场景描述

```
无论你是什么结局，在最后，你都会看到小林。

他站在公司楼下，等你。

"前辈……"

他的眼睛有些红。

"我要走了。"

他看着你：

"前辈，我……我拿到了另一家公司的offer。
 是一家小公司，待遇一般。
 但我想……去试试。"

他苦笑：

"我在这里，找不到自己的位置。
 每天都活得很压抑。
 与其这样，不如换个环境。"

他看着你：

"前辈，谢谢你当初……愿意帮我。"
```

### 小林的对照

```
【你的结局 + 小林的对照】

如果你成功了：
小林站在你面前，有些拘谨。

"前辈，恭喜你……"

他的眼神里有羡慕，但也有释然：

"你走得比我远。
 但我不会后悔。
 因为我知道，我选择了自己的路。"

小林成为你人生的过客。
你们各自走向不同的方向。


如果你失败了：
你们一起站在公司门口，相对无言。

"前辈，我们接下来……怎么办？"

他看着你：

"也许……这里从一开始就不适合我们。"

你们成为彼此的镜子。
 你们都是这个环境的受害者。


如果你离开了：
半年后，你听说小林"混得不错"。

他换了一家小公司，做得风生水起。

"他找到了自己的路。"
```

## 节拍3：李总监/王副总对话

### 李总监对话（高信任）

```
"小林，这次的事……不是你的问题。"

他看着你：

"王浩的事，我知道。
 张姐的事，我也知道。"

他叹了口气：

"职场就是这样。
 有能力的人，不一定能留下来。
 能留下来的人，不一定有能力。"

他站起来：

"但你……让我看到了另一种可能。
 不是妥协，而是坚持。
 不是随波逐流，而是保持自我。"

他伸出手：

"如果你愿意回来，我可以给你推荐信。
 如果你不想回来……祝你好运。"
```

### 李总监对话（低信任）

```
"小林……"

他看着你，表情复杂。

"你知道为什么你没有留下来吗？"

你沉默。

"因为你还不够强。
 不是能力不够强。
 是……你还不够懂这个游戏。"

他站起来：

"职场不是做题。
 不是你做对了，就能得分。
 有时候，做对了，也可能丢分。"

他看着你：

"也许……换个环境，你会更好。
 但记住，不是所有人都适合这里。"
```

## 节拍6：代价揭示

### A类结局：成功的代价

```
【揭示】

你成功了。

但你失去了什么？

- 你把所有时间都给了工作
- 和朋友的关系淡了
- 你开始理解李总监当年的选择
- 你的头发白了几根
- 你忘记了上一次陪家人是什么时候
- 你变成了当初自己最不想成为的人

但你也得到了什么？

- 尊重
- 地位
- 能力
- 成长
- 还有一些真正的朋友

【代价与获得】

成功没有完美的。
每一种成功，都有代价。
关键是，你愿意承担吗？
```

### B类结局：稳定的意义

```
【揭示】

你留下来了。

但你得到了什么？失去了什么？

- 你获得了稳定
- 但你也放弃了冒险
- 你可能会有遗憾，但也不后悔
- 你学会了和自己和解
- 你找到了工作与生活的平衡
- 你开始理解：成功不是只有一种方式

【稳定与代价】

稳定也是一种成功。
不是每个人都要站在最高处。
关键是，你想要什么？
```

### C类结局：离开的反思

```
【揭示】

你离开了。

但这意味着什么？

- 你付出了代价
- 但你也学到了教训
- 你可能失去了工作，但找到了自己
- 你可能失败了，但找到了方向
- 你可能被迫离开，但开始了新的故事

【离开的意义】

离开不是失败。
有时候，放手也是一种勇气。
关键是，你从中学到了什么？
```

## 节拍7：价值观终极选择

### 场景描述

```
无论你是什么结局，在最后，你都要回答一个问题。

这个问题，从第一章就开始了：
"在职场，我该成为什么样的人？"

张姐说："没有永远的敌人和朋友，只有永远的利益。"
李总监说："证明自己的价值。"
王浩说："你不抢，就没得吃。"
陈叔说："不要走我的老路。"

但你呢？

你想成为什么样的人？
```

### 价值观选择

```
【终极选择】

A. 坚持原则，不妥协
   "不管环境怎样，我要做我认为对的事。"

B. 适应规则，但不放弃底线
   "我理解这个游戏，但我不会变成我最讨厌的人。"

C. 学会妥协，保护自己
   "在职场，生存是第一位的。"

D. 找到平衡，不走极端
   "成功有很多种方式，不是只有一条路。"
```

### 选择的后果

```
【选择的意义】

A："坚持原则，不妥协"
→ 你可能失去更多
→ 但你保持了自我
→ 你会后悔吗？

B："适应规则，但不放弃底线"
→ 你学会了生存
→ 但你也保持了自我
→ 这是最好的平衡吗？

C："学会妥协，保护自己"
→ 你活得更好
→ 但你变成了另一个人
→ 值得吗？

D："找到平衡，不走极端"
→ 你有了自己的路
→ 但你也在妥协
→ 这就是成长吗？
```

## 节拍8：终场画面

### 开场回顾

```
【第一章开场】

入职第一天早上8:30。

你站在公司楼下，抬头看着这栋30层的写字楼。

手心有点出汗。

你深吸一口气，走进旋转门。

"这是我职场人生的第一天。
 我告诉自己，要好好表现。
 但我不知道该成为什么样的人。"

【开场基调】
迷茫 + 害怕 + 不知道
```

### 终场画面（优化）

```
【第五章终场】

无论你是什么结局。

最后，你都会站在公司门口。

或离开，或留下。

但你的眼神，和第一天不一样了。

"这是我职场人生的一个阶段。
 不是终点，是过程。

我学到了什么？
 我得到了什么？
 我失去了什么？

张姐说，没有永远的朋友。
 李总监说，证明自己的价值。
 陈叔说，不要走他的老路。

但我说……

我是我自己。
 我选择了我想成为的人。
 不管这个选择是对是错。
 这就是代价。
 这就是成长。"
```

```
【开场与终场】

开场（第1章）：
"我不知道该成为什么样的人。"
```
