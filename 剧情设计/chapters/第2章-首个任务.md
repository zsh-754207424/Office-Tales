## 节拍1：任务开始

### 场景描述

```
入职第三天。

早上9点，李总监把你叫到办公室。

"小林，上次那个PPT，你研究得怎么样了？"

他看着你，等待你的回答。

你把修改后的PPT给他看。
他快速浏览了一遍，点了点头。

"不错，有进步。"

他打开电脑，调出一份文件。

"现在有个任务给你。
 公司准备推出一款新产品，需要做市场分析报告。
 这是相关资料，你拿去研究，一周后给我一份完整的分析报告。"

他把一个U盘推到你面前。

"记住，我看重的是数据和逻辑，不是漂亮的话术。"

他看了一眼手表：
"有问题随时来找我。但不是大问题，不要浪费我时间。"
```

### 玩家选项

```
A. 询问具体要求
   "李总，这个报告您希望侧重哪个方面？"

B. 表示没问题
   "好的，李总，我会尽快完成的。"

C. 确认截止时间
   "好的，一周后给您。请问是周五下班前还是周一早上？"
```

## 节拍2：完成任务 + 娱乐游戏展示（优化）

### 场景描述

```
接下来几天，你全身心投入任务。

这是你的"娱乐游戏"时刻。

【娱乐游戏1：市场调研】

你开始收集竞品资料。

你发现：
- A品牌的市占率最高，但价格也最高
- B品牌价格低，但功能单一
- C品牌是新兴势力，增长迅速

你在分析中找到了自己的洞察：

"智能办公软件市场的痛点不是价格，
 是用户习惯的改变。
 真正的竞争对手不是其他软件，
 而是'不用软件'的选择。"

这个洞察让你兴奋。

你加班到晚上11点，把分析写成了报告。

【娱乐游戏2：克服困难】

期间，你遇到了问题。

有些数据找不到官方来源。

但你没有放弃。

你用公开报道、行业报告、甚至用户评价来交叉验证。

最终，你构建了一套完整的分析框架。

张姐看到你加班，给你带了杯咖啡：

"小林，第一周就这么拼，不容易。"

她看了你的报告，点了点头：

"这个思路很好。
 李总喜欢有洞察的报告。
 你会有机会的。"

【娱乐游戏3：完成任务】

周日晚上10点，你终于完成了报告。

你检查了三遍，确保没有问题。

报告标题：
"智能办公软件市场分析报告_v1.0"

你很满意。

你知道，这是你能力的证明。

这是你在这个世界的"娱乐游戏"：
做有价值的事，证明自己的价值。
```

### 玩家选项（遇到问题时的处理）

```
A. 向张姐请教
   （获得帮助，好感+5）

B. 继续自己研究
   （独立完成，展示能力）

C. 向李总监汇报进度
   （展示主动性）

D. 找王浩帮忙
   （可能有陷阱）
```

## 节拍3：中点 - 伪胜利（新增）

### 场景描述

```
周一早上。

你正准备去汇报。

但李总监先找到了你。

"小林，你来一下。"

你有些意外。

他带你到小会议室，关上门。

"小林，你的报告我看过了。"

他看着你：

"你用的是波特五力模型做的分析？"

你点头。

"嗯，这个框架用得不错。
 但你后面加的那个洞察——'真正的竞争对手是选择不用软件'——是你自己想的？"

你点头。

他沉默了一会儿，然后说：

"小林，你知道为什么我让你做这个报告吗？"

你摇头。

"因为我想看看你有没有自己的思考。
 很多人只会套框架，没有洞察。
 但你有。"

【中点：伪胜利】

这一刻，你以为你站稳了脚跟。

李总监继续说：

"你的能力，我看到了。
 好好干，以后会有机会。"

你感到一种巨大的满足感。

【内心独白】
"李总说我有能力。
 我证明了自己的价值。
 在这个环境里，我是可以的。"

这一刻，你获得了虚假的胜利。

你不知道的是——
这只是暴风雨前的宁静。

王浩已经开始行动了。

【伪胜利的代价】

你以为自己安全了。
但危险正在逼近。

这就是"伪胜利"：
看起来像成功，但不是真正的成功。
它让玩家放松警惕，然后……
然后"坏蛋"来了。
```

### 中点效果

```
【伪胜利效果】
- 李总监信任+15
- 玩家感到满足
- 玩家放松警惕
- 为后面的"一无所有"增加反差

【坏蛋逼近的铺垫】
但就在这个"胜利"的同一天……
王浩开始行动了。
```

## 节拍4：王浩观察铺垫（因果优化）

### 场景描述

```
周一早上10点。

你刚从李总监办公室出来，感到满足。

但你没有注意到……

王浩在茶水间看到了这一幕。

他看到你和李总监单独谈话。
他看到你出来时脸上的笑容。
他看到李总监拍了拍你的肩膀。

【王浩的视角】

王浩拿起手机，发了一条消息：

"……他得到李总认可了。"

王副总回复：
"嗯。继续观察。"

【王浩的行动】

王浩放下手机，走到你工位附近。

他"刚好"路过，看了一眼你的屏幕。

你正在整理报告的最终版。

王浩停下脚步。

"哟，竞品分析啊？"

他凑近了一些：

"你这个数据来源是哪里？
 我之前做过类似的，
 有些官方数据不太好找……"

你注意到他看得很仔细。

他若有所思地点了点头：

"嗯，这个思路不错。
 市场容量那块，你用的什么模型？"

你简单解释了一下。

他又看了几秒，然后拍拍你的肩膀：

"加油，新人都是这么过来的。
 有问题可以问我。"

然后他就走了。

你继续工作，没有多想。

【玩家视角的内心独白】
"王浩'刚好'看了我的报告。
 他看起来很热心。
 但……总觉得哪里不对。"

【BS2视角的真相】

王浩在评估：
- 这个新人的能力怎么样？
- 他的报告值不值得"用"？
- 怎么"用"最安全？

周三他"刚好"路过。
周五他会"刚好"有资料。
周一他会"刚好"帮你提交。

不是巧合，是他计划的一部分。

【悬念设计：戏剧反讽】

玩家现在知道：
王浩是敌人。
他正在计划什么。
但玩家不知道他会怎么做。

这就是"戏剧反讽"：
读者知道危险在逼近，但角色不知道。
这增加悬念，让玩家紧张。
```

## 节拍5：王浩的帮助（可疑的"好意"）

### 场景描述

```
周五下午。

你正在修改报告，准备最终版。

王浩走过来。

他看了一眼你的屏幕。

"哟，市场分析报告啊？"

他很自然地拉了把椅子坐下。

"你这个竞品分析做得不错，
 但数据来源可以再补充一下。
 我这有些资料，要不发给你参考参考？"

他的态度很热情。

但你想起：
- 周三他"刚好"看到你的屏幕
- 周一他"刚好"看到李总监和你谈话
- 现在他又"刚好"有资料

【因果逻辑】

这不是巧合。

你开始怀疑他的动机。

但你没有证据。

他可能只是热心。
也可能……

他可能有别的目的。

你想起张姐的话：
"王浩那个人，表面热情，你要小心。"

他把手机递过来：

"来，我扫一下你微信，我发给你。"

【内心独白】
"他的好意……是真的吗？
 还是一个陷阱？
 但如果拒绝他，会不会显得太防备？
 在这个环境里，太防备是不是也不好？"
```

### 玩家选项

```
A. 接受帮助，扫码加微信
   "谢谢浩哥，有资料当然好。"

B. 礼貌拒绝
   "谢谢，但我想自己试试。"

C. 接受但保持警惕
   "谢谢，不过我自己有资料，你发的我参考一下。"

D. 直接拒绝
   "不用了，我自己能搞定。"
```

### 数值影响

| 选项 | 王浩敌意 | 备注 |
|------|----------|------|
| A | -5 | 可能获得"帮助"，但有风险 |
| B | +0 | 保持距离 |
| C | -5 | 接受部分帮助，但有限度 |
| D | +5 | 可能得罪他 |

## 节拍6：抢功事件（第一扇门）

### 场景描述

```
周日晚上。

你终于完成了报告的最终版。

你检查了三遍，确保没有问题。

你把报告命名为：
"智能办公软件市场分析报告_v1.0"

这是你的心血。
这是你能力的证明。
这是你在这个世界的立足点。

【内心独白】
"李总说我有能力。
 我证明了自己。
 明天，我会让所有人看到。"

你不知道的是……

王浩已经计划好了一切。

---

周一早上。

你正准备去汇报。

王浩突然走过来。

"小林，你的报告做完了吧？"

你点头。

他笑着说：

"我正好要去李总办公室，顺路帮你带过去？
 你不用跑一趟了，我帮你交。"

你犹豫了一下。

你想起：
- 周三他"刚好"看过你的屏幕
- 周五他"刚好"有资料要给你
- 现在他"刚好"要帮你提交

一切都太巧了。

但你还是没有多想。
你把U盘递给了他。

【半小时后】

群里突然弹出一条消息。

李总监："@王浩 市场分析报告收到，分析得很到位。
         下午开会你来汇报。"

你愣住了。

你看到王浩在群里回复：

"谢谢李总，这是我应该做的。"

【一无所有：身份死亡】

你的报告，被抢走了。

你的成绩，被人拿走了。

你以为自己站稳了脚跟。
但一切都是假的。

这就是"一无所有"：
你失去了一切——身份、成就、证明。

---

王浩经过你工位时，笑着说：

"谢了啊，小林。"

然后他若无其事地走了。

周围同事的反应：
- 张姐看了你一眼，什么都没说
- 其他同事似乎没有注意到
- 小林欲言又止，最后还是没说
```

### 🚪第一扇门触发

```
【门的条件】
王浩把你的报告交给领导
领导表扬了王浩
你的名字没有被提到

【因果已铺垫】
王浩从周三就开始计划：
- 周三：观察你的能力
- 周五：建立联系
- 周一：抢走成果

不是巧合，是他计划的一部分。

【中点与一无所有的对比】

中点（伪胜利）：
李总监私下认可你 → 你以为自己成功了

一无所有：
王浩抢走你的报告 → 你失去了一切

这就是"赏金提高"：
你以为自己赢了，但真正的挑战才刚刚开始。

【门的性质】
关系不可逆转转折
故事只能向前
```

## 节拍7：应对选择（关键选择）

### 场景描述

```
你坐在工位上，看着群里的消息。

手指在键盘上发抖。

【愤怒、无助、憋屈】

你知道这个报告是你做的。
你知道王浩做了什么。
你知道周围的人在看着你的反应。

你也知道……
这不是巧合。
从周三开始，他就在计划了。

张姐从你身边走过，停顿了一下。
她小声说：
"你看着办。但如果要找，最好有证据。"

小林也走过来，欲言又止：
"前辈，我……我上次看到他……"

他压低声音：
"他好像在提交之前，把你的报告打印出来看了很久。"

你开始回忆：
- 周三他"刚好"看你的屏幕
- 周五他"刚好"有资料
- 周一他"刚好"帮你提交
- 你的U盘在他手里大概有半小时

你打开电脑，检查了一下：

文件修改记录显示：
"智能办公软件市场分析报告_v1.0"
最后一次修改时间：周日晚上10:23
但还有一次访问记录：周一早上8:47
访问者：王浩的电脑

你有证据了。

但这只是证明他访问过文件。
要怎么证明……是他拿走的呢？

【内心独白】
"李总说我有能力。
 但现在，没有人知道那是我做的。
 我以为我站稳了脚跟。
 原来……一切都是假的。

这就是职场吗？
 这就是'证明自己的价值'吗？

我该怎么办？"
```

### 玩家选项

```
A. 当面质问
   找到王浩，直接摊牌：
   "王浩，那个报告是我做的。你为什么要这么做？"

B. 沉默接受
   咽下这口气，假装什么都没发生。
   （可能在等待机会）

C. 私下找领导
   单独找李总监，把证据给他看：
   "李总，有些事情我想跟您反映……"

D. 收集更多证据，保存记录
   暂时不动声色，保留所有证据。
   等待更好的时机。
```

### 数值影响

| 选项 | 王浩敌意 | 李总监信任 | 其他影响 |
|------|----------|------------|----------|
| A | +30 | +5（敢担当） | 公开对立 |
| B | +10 | +0 | 他觉得你好欺负 |
| C | +50 | +10（但有风险） | 彻底撕破脸 |
| D | +5 | +0 | 保留筹码 |

### 场景反应

```
【选A后的场景】
你找到王浩：
"王浩，那个报告是我做的。"
王浩一愣，然后笑了：
"什么你的报告？李总交给我做的。"
"我有证据。"
你把修改记录给他看。
他脸色变了：
"你想怎样？"
"我只是想知道为什么。"
他沉默了一会儿：
"职场就是这样，你不抢就没得吃。"
然后他转身走了。
从那以后，你们的关系彻底破裂。
```

## 节拍8：拜访陈叔 + 交换条件

### 场景描述

```
抢功事件处理完后。

你在茶水间遇到了那个头发花白的老人。

他正在泡茶，动作很慢，很专注。

你之前听张姐提过他：
"老陈啊？他以前是总监，后来不知道为什么被边缘化了。不用理他。"

但你现在有些迷茫。

他抬起头，看了你一眼：

"年轻人，有心事？"

你犹豫了一下。

【交换条件】

你坐下来，决定和他聊聊。

你把王浩抢功的事告诉了他。

他听完，叹了口气：
"这种事，我见得多了。"

他看着你：
"你想知道真相？"

你点头。

"我可以告诉你一些事。
 但你要答应我一件事。"

他看着你：

"将来如果有人问你，这些事是谁告诉你的……
 你说不记得了。"

他放下茶杯：

"也许是在茶水间听到的。
 也许是网上看到的。
 也许是张姐，李总监……任何人。
 总之，不要提我的名字。"

他叹了口气：

"我已经边缘化了，但我还想保住这份工作。
 如果被知道我给你透露这些……
 我可能连现在的位置都没有了。"

他伸出手：

"能答应吗？"
```

### 玩家选项

```
A. 答应，主动攀谈
   "陈叔，我答应您。您愿意告诉我这些，我很感激。"
```
