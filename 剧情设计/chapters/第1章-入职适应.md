# 第1章-入职适应（优化版）

## 章节概述

```
章节：第1章
时长：入职第1天
核心目标：适应环境，获得初步认可
核心冲突：同期竞争、领导观察、融入团队
关键选择：4个
章节结局：建立初始人际关系 + B故事铺垫

【BS2结构对应】
- 开场画面（第1页）：设定基调，展示职场新人的紧张与期待
- 主题呈现（第5页）：提出"职场中，该如何自处？"的问题
- 铺垫（第1-10页）：介绍所有人物，建立世界
- 推动/催化剂（第12页）：第一次汇报让玩家感受压力
- 争执（第12-25页）：玩家决定如何应对

【优化内容】
+ 增加小林作为B故事的完整铺垫
+ 增加主角内心独白（让玩家思考主题）
+ 增加"被迫在一起"的冲突（王浩同期入职）
```

## 节拍1：入职手续（认识小刘HR）

### 场景描述

```
早上9点，你来到公司。

前台带你到人力资源部办理入职手续。

一个穿着职业装的女孩站起来，笑容标准但有距离感：
"你好，我是HR刘思琪，叫我小刘就行。来，先填一下这些表格。"

她把一叠文件推到你面前，然后低头继续工作。

你填表的时候，注意到她的桌上摆满了各种文件夹。
她不时看一眼电脑屏幕，表情专注。

突然，她的手机响了。

她看了一眼来电显示，脸色微微变了。

"不好意思，我接个电话。"

你听到她小声说：
"……是，我知道了。他什么时候……好，好的。"

她挂了电话，表情恢复了平静。

但你注意到，她的手指在轻轻发抖。

【内心独白】
"她的电话内容……我不知道是什么。
 但那一刻，她看起来也很脆弱。
 每个人都有自己的难处。"
```

### 场景信息

```
【小刘HR登场】
职位：HR专员
态度：客气但保持距离
观察点：她有自己的压力，但职业化处理

【可获取信息】
- 入职流程
- 公司基本制度
- 她的第一印象

【隐藏信息（玩家不知道）】
- 小刘两年前差点被裁
- 李总监保了她
- 这是她为什么愿意帮主角的原因
```

### 玩家选项

```
A. 趁填表时和她聊几句
   "小刘姐，公司平时几点下班啊？"
   （友好但保持距离）

B. 安静填表，不打扰她
   （观察她的反应）

C. 注意到她的紧张，假装没看到
   （职业敏感度）
```

## 节拍2：部门介绍（认识李总监）

### 场景描述

```
填完表格，前台带你到15楼，市场部。

一个中年男人从办公室走出来。
他穿着深色西装，头发短促，眼神锐利。

"你好，我是市场部总监李建华。"

他的声音不大，但有一种不怒自威的气场。

他看了一眼手表：

"现在9:15。你跟我来，给你介绍一下部门。"

他带你走过办公区：

"这是张雨晴，高级专员。"
张姐站起来，礼貌地笑了笑。

"这是王浩，新人，比你早来两周。"
王浩笑着点头："欢迎欢迎。"

"这是……"李总监看了看角落。
一个戴着眼镜的年轻人正在埋头看资料。
"小林，实习生。别打扰他。"

【内心独白】
"三个人，三种状态。
 张姐专业，王浩热情，小林紧张。
 我会变成谁？"
```

### 李总监的观察

```
李总监带你回到他的办公室。

"小林，我知道你是新人。
 但我想先跟你说清楚几件事。"

他看着你：

"第一，我看重结果，不看过程。
 第二，有问题随时找我，但要有解决方案。
 第三……"

他停顿了一下：

"在这里，每个人都要证明自己的价值。
 不是我觉得你有价值就有价值，
 是你做出成绩，才算有价值。"

【李总监的核心信息】
"证明自己的价值"
这句话是第一章的主题核心

【内心独白】
"证明自己的价值……
 什么是价值？
 是业绩？是人际关系？还是不被裁掉？"
```

### 玩家选项

```
A. 认真记下他的要求
   "好的，李总，我会努力的。"

B. 询问具体标准
   "李总，您说的价值，具体是指什么？"

C. 表示压力
   "李总，我会尽力做好。"
```

## 节拍3：认识同事（张姐、王浩、小林）

### 场景描述

```
离开李总监办公室，你回到工位区。

张姐走过来，笑容温和：

"小林，我是张雨晴，大家都叫我张姐。
 有什么不懂的可以问我。"

她压低声音：

"新人嘛，都会紧张，慢慢来就好。"

【张姐初印象】
- 热情但不越界
- 有经验但不炫耀
- 她的话里有话

---

这时，一个年轻人走过来。

他看起来比你大一点，阳光帅气，笑容标准：

"小林吧？我是王浩，也是新人，比你早来两周。
 以后咱们多交流。"

他的热情有些过头，但看起来很真诚。

【王浩初印象】
- 热情，主动
- 但笑容太标准，像练习过无数遍
- 眼神里有一种评估的意味

---

你注意到角落里有个人。

他戴着眼镜，很瘦，在埋头看资料。
看起来比你还紧张。

小林。

他是实习生，比你早来一个月。
但看起来完全没有适应。

【小林初印象】
- 紧张、迷茫
- 缺乏自信
- 不知道该怎么办

【B故事铺垫】
小林是你的镜像：
- 你们都是新人
- 你们都迷茫
- 但你们选择了不同的应对方式
```

### B故事暗示

```
【内心独白】
"小林看起来比我更紧张。
 他让我想起自己刚踏进公司的那一刻。

如果我们都没有方向……
 那谁才是对的？
 我们都会变成什么样？"
```

## 节拍4：午餐选择（关键选择1）

### 场景描述

```
快到中午了。

张姐收拾东西，准备出门。

"小林，中午一起吃饭吗？我带你去食堂。"

就在这时，王浩也走过来了。

"张姐，中午一起啊？难得今天新人齐。"

他看向你：

"小林，一起吧？新人就该多交流。"

张姐看了看你，没有说话。

王浩的笑容依然标准。

但你注意到……
张姐的眼神里有一丝不易察觉的复杂。

【冲突来源】
王浩的"热情"让你有些不安
张姐似乎想说什么，但没有说

【被动在一起的感觉】
你不想去，但不知道怎么拒绝
你不想得罪任何人，但不得不选择
```

### 玩家选项

```
A. 接受邀请，和张姐、王浩一起吃饭
   "好啊，谢谢张姐。"
   
   【内心独白】
   "我选择融入。
    虽然感觉哪里不对，但先站稳脚跟。"

B. 单独和张姐吃
   "张姐，我中午有点事，改天请您？"
   
   【内心独白】
   "我不想和王浩走太近。
    但这样做，对吗？"

C. 单独和王浩吃
   "王浩，一起吗？"
   
   【内心独白】
   "他在主动接近我。
    也许……利用这个机会？"

D. 说自己有事，自己吃
   "我中午有点东西要处理，你们先走。"
   
   【内心独白】
   "我谁都不想选。
    我想自己先看看清楚。"
```

| 选项 | 张姐好感 | 王浩敌意 | 小林好感 | 备注 |
|------|----------|----------|----------|------|
| A | +5 | -5 | +0 | 三人聚餐，但感觉微妙 |
| B | +10 | +5 | +0 | 选择张姐，但可能得罪王浩 |
| C | -5 | -10 | +0 | 选择王浩，张姐会注意到 |
| D | +0 | +5 | +0 | 保持距离，但可能显得不合群 |

## 节拍5：第一次汇报（关键选择2）

### 场景描述

```
下午2点，李总监把你叫到办公室。

"小林，下午有个客户要来。
 我准备了一份产品介绍PPT。
 你看一下，有什么想法。"

他把电脑转向你。

PPT内容是一份智能办公软件的产品介绍。
数据很多，但逻辑有些混乱。

李总监看着你：

"说说你的想法。"

【这是你第一次面对"证明价值"的压力】

【观察点】
- PPT确实有问题
- 李总监在评估你
- 你的回答会影响他对你的判断
```

### 主题思考（关键）

```
【内心独白】
"张姐说'不要得罪不该得罪的人'。
 李总监说'要证明自己的价值'。

这两句话是矛盾的：
- 如果我直接指出问题，会不会得罪李总监？
- 如果我附和，会不会显得没能力？
- 什么才是'正确的'应对？"

这是第一章的主题核心问题：
"在职场，我该怎么自处？"
```

### 玩家选项

```
A. 直接指出问题
   "李总，我觉得第5页的数据和第7页有矛盾，
    还有整体逻辑可以更清晰……"
   
   【代价】
   有主见，但可能显得太直接

B. 委婉提建议
   "李总，我对产品不太了解。
    但如果数据更清晰，可能会更好……"
   
   【代价】
   谦虚但缺乏主见

C. 表示需要时间
   "李总，能给我几分钟仔细看看吗？"
   
   【代价】
   稳妥，但可能显得没想法

D. 简单附和
   "看起来很专业。我还需要多学习。"
   
   【代价】
   没态度，但安全
```

### 数值影响

| 选项 | 李总监信任 | 备注 |
|------|------------|------|
| A | +10（风险） | 有主见，但可能太直接 |
| B | +5 | 谦虚但有想法 |
| C | +5 | 稳妥 |
| D | -5 | 没态度 |

## 节拍6：遇到小林（关键选择3）+ B故事铺垫

### 场景描述

```
从李总监办公室出来，你回到工位区。

小林走过来，手里拿着一个文件夹。

"小林……不对，前辈……不好意思，我是说……"

他紧张得说话都结巴了。

"我看你刚从李总办公室出来。
 他是不是布置了什么任务？"

他推了推眼镜：

"我下午也要去汇报。好紧张……"

他小声说：

"前辈，我……我能请教你吗？
 我不知道该怎么汇报。
 我怕说错话……"

【B故事的关键铺垫】

小林是"另一个可能的你"：
- 你们都是新人
- 你们都迷茫
- 你们都面临"如何证明价值"的问题

他对你的态度，暗示了你将来会变成什么样。

【内心独白】
"小林的眼睛里，有一种我很熟悉的东西。
 迷茫、害怕、不知所措。

但我又能帮他什么？
 我自己都不知道路在哪里。"
```

### B故事深化

```
你看着小林，想起了什么。

他让我想到入职第一天的自己。
如果我当时也这样紧张……
如果没有人帮我……

【小林的处境】
- 实习生，地位更低
- 没有人帮他
- 他不知道该问谁

【你选择帮/不帮，影响小林的命运】
- 如果帮他，他可能成为你的盟友
- 如果不帮，他可能成为另一个"迷失的灵魂"
```

### 玩家选项

```
A. 热心帮助
   "来，我们一起看看你的材料。
    我刚才也汇报了，有些经验。"
   
   【小林好感+20】
   【你选择成为"帮助者"】
   
   【内心独白】
   "我帮了他。
    也许有一天，也会有人帮我。
    在这个环境里，善意是存在的。"

B. 有限帮助
   "李总喜欢数据说话，你多准备一些数据。"

   【小林好感+10】
   【你选择"有分寸"的帮助】
   
   【内心独白】
   "我帮了他一点。
    但我不能太多卷入。
    我自己都还在摸索。"

C. 礼貌拒绝
   "我也是新人，你去看看张姐吧。"
   
   【小林好感+0】
   【你选择"自保"】
   
   【内心独白】
   "我没有帮他。
    不是不想，是不敢。
    我连自己都顾不过来。"

D. 冷淡回应
   "每个人情况不一样，你自己看着办吧。"
   
   【小林好感-10】
   【你选择"自我保护"】
   
   【内心独白】
   "我拒绝了他。
    不是因为我不在乎。
    是因为……我不知道怎么在乎。"
```

### 数值影响

| 选项 | 小林好感 | B故事线 | 备注 |
|------|----------|----------|------|
| A | +20 | 深度连接 | 成为真正的盟友 |
| B | +10 | 有限连接 | 保持距离 |
| C | +0 | 无连接 | 平行线 |
| D | -10 | 负面连接 | 可能成为对手 |

## 节拍7：张姐的忠告 + 主题强化

### 场景描述

```
下午6点，大部分人开始收拾东西。

张姐走过你工位，停下脚步。

"小林，第一天感觉怎么样？"

她压低声音：

"我多跟你聊几句。"

她看了看四周：

"小林，我跟你说几件事，你记一下：

第一，王浩那个人……表面热情，但你小心点。
 他上周把一个实习生的方案拿过去汇报了。
 那个实习生差点被辞。

第二，有什么不懂的可以问我。
 但我也有我的立场。
 我是李总这边的人。
 我帮你的前提是……你不给我惹麻烦。

第三……"

她停顿了一下：

"在这里，没有永远的敌人，也没有永远的朋友。
 只有永远的利益。
 你要搞清楚，谁是你的敌人，谁可以利用。"

【主题强化】

张姐的话，是张姐版本的"如何自处"：
- 在这里，没有对错，只有利益
- 善良是有条件的
- 生存是第一位的

【你的选择】

你会相信她的话吗？
你会变成她那样的人吗？

【内心独白】
"张姐的话很现实。
 现实到让我有些害怕。

但这就是职场吗？
 这就是'证明自己的价值'的意思吗？

也许……

也许有一天，我也会变成这样。
 带着善意为别人好，但有底线。
 保护自己，但帮助值得帮助的人。

这就是成长吗？

我不知道。

但我知道，我不想变成一个只关心利益的人。
 我想保有一些东西。
 即使在这个环境里。"
```

## 第一天结束

```
下午6:30。

你坐在工位上，整理一天的经历。

入职第一天结束了。

你认识了：
- 李总监（严格但公正）
- 张姐（热情但有条件）
- 王浩（热情但可疑）
- 小林（紧张但真诚）
- 小刘HR（专业但有自己的难处）

你学到了：
- 要证明自己的价值
- 要小心不该得罪的人
- 没有永远的敌人和朋友

但你还不确定：
- 什么是"正确的"职场方式
- 你会成为什么样的人
- 你能不能在这里生存下去

明天，新的挑战在等着你。
```
