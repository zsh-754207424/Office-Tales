# 剧情套路设计手册

## 核心思路

> **好的套路不是老套，而是"意料之外，情理之中"**
>
> 本游戏使用**经典情节套路**增强戏剧张力，同时确保每个套路都有**合理的因果铺垫**

---

## 一、已使用的经典套路

### 套路1：镜像角色（Mirror Character）

```
【定义】
一个与主角相似但做出不同选择的人物
通过对比揭示主题

【本游戏应用】
小林 = 另一个可能的"你"
- 如果你没有主动，会变成他
- 他的结局是你的对照

【效果】
让玩家看到选择的后果
"如果我当初……"
```

### 套路2：两扇门（Two Doors）

```
【定义】
不可逆转的转折点，让故事只能向前

【本游戏应用】
第一扇门：抢功事件（第2章）
第二扇门：裁员名单（第4章）

【效果】
增加选择的重量
"没有回头路"
```

### 套路3：智者导师（Mentor Figure）

```
【定义】
提供智慧但自身有缺陷的角色

【本游戏应用】
陈叔
- 知道真相，但自己也曾是失败者
- 他的建议有价值，但代价是看清残酷现实

【效果】
让信息更有重量
"说这话的人，自己也经历过"
```

### 套路4：致命诱惑（Devil's Bargain）

```
【定义】
给你想要的，但要你付出代价

【本游戏应用】
王副总的拉拢
- 承诺保住你，但要你站队
- 代价是价值观的妥协

【效果】
让"反派"更有吸引力
增加选择的道德模糊性
```

### 套路5：倒计时（Countdown）

```
【定义】
有时间限制的紧迫感

【本游戏应用】
裁员名单公布前的等待
- 玩家知道时间在流逝
- 每个选择都在"最后机会"中做出

【效果】
增加紧张感
"时间不多了"
```

## 二、需要加强的经典套路

### 套路6：盟友背叛（Ally Betrayal）

```
【定义】
最信任的人突然成为障碍

【计划加入】
张姐的动摇：
- 第3章中期，王浩开始接近张姐
- 张姐对你说："你不要让我为难"
- 她被迫在你们之间选择

【为什么加】
打破"盟友=永远可靠"的预设
增加不确定性
让玩家更珍惜真正站在你这边的人
```

### 套路7：真相揭露（Reveal）

```
【定义】
揭示角色不愿提起的过去

【计划加入】
陈叔的过去：
- "你知道为什么我知道这么多吗？"
- "十年前，我也是这样被卷进来的"
- "我选了王副总那边。然后……我失去了一切"

【为什么加】
让陈叔更立体
让主角的选择更有重量
"有人走过这条路，失败了"
```

### 套路8：最后机会（Last Chance）

```
【定义】
在最后时刻提供转机

【计划加入】
裁员前夜：
- 小刘HR深夜发消息
- "名单还有变数。如果你能做成一件事……"
- 提供最后一个选择机会

【为什么加】
增加紧张感
给玩家控制感
"还有机会，但只有一次"
```

### 套路9：镜像对话（Mirrored Dialogue）

```
【定义】
用另一个角色的口说出主角的心声

【计划加入】
第5章，小林找你：
- "前辈，我拿到另一家公司的offer了"
- "你觉得我该走还是该留？"
- 你看着他，仿佛看到曾经的自己

【为什么加】
让玩家反思自己的选择
用第三人称审视自己
"你想对他说的话，其实是你想对自己说的"
```

### 套路10：交易式抉择（The Deal）

```
【定义】
在两个都不完美的选项中抉择

【计划加入】
第4章：
- "你有两个选择"
- "A：让王浩走，但你要帮他背一个锅"
- "B：你留下，但王浩也留下，你们继续对抗"

【为什么加】
没有完美选项
每个选择都有代价
"成年人的世界没有对错"
```

## 三、套路应用检查清单

### 每个章节需要的套路

```
第1章：入门
└─ 套路1（镜像角色）✓ - 小林登场

第2章：成长
├─ 套路3（智者导师）✓ - 陈叔可选接触
├─ 套路4（致命诱惑）✓ - 王浩的"帮助"
└─ 套路2（两扇门）✓ - 抢功事件

第3章：抉择
├─ 套路6（盟友背叛）← 需要加入
├─ 套路4（致命诱惑）✓ - 王副总拉拢
└─ 套路5（倒计时）✓ - 裁员风声

第4章：危机
├─ 套路7（真相揭露）← 需要加入
├─ 套路8（最后机会）← 需要加入
├─ 套路10（交易抉择）← 需要加入
└─ 套路2（两扇门）✓ - 裁员名单

第5章：结局
├─ 套路9（镜像对话）✓ - 小林结局对照
└─ 套路1（镜像角色）✓ - 代价揭示
```

### 每个角色需要的套路

```
李总监：套路3（导师） + 套路7（过去）
王浩：套路4（诱惑） + 套路6（背叛）
陈叔：套路3（导师） + 套路7（过去）← 加强
张姐：套路6（背叛）← 加入
小刘HR：套路8（最后机会）← 加入
小林：套路1（镜像）+ 套路9（对话）← 加强
秦总：套路7（过去） + 代价揭示
王副总：套路4（诱惑） + 套路10（交易）
```

## 四、套路使用原则

### ✅ 应该做的

1. **每个套路都要有因果铺垫**
   - 王浩的背叛要有前兆
   - 陈叔的过去要有暗示

2. **让套路服务于主题**
   - "你想成为什么样的人"是核心
   - 每个套路都要强化这个主题

3. **给玩家选择权**
   - 即使有套路，也要让玩家选择应对方式
   - 不要让套路"发生"在玩家身上

### ❌ 不应该做的

1. **不要让套路太明显**
   - 反派不要太像反派
   - 盟友不要太可靠

2. **不要让巧合成为主要推动力**
   - 每个事件都要有因果
   - 套路是催化剂，不是解决方案

3. **不要让套路破坏情感真实**
   - 即使是套路，人物反应也要真实
   - 不要为了戏剧性牺牲合理性

## 五、套路与选择的关系

```
【套路是框架，选择是内容】

套路（框架）：
- 盟友背叛 → 你怎么应对？
- 致命诱惑 → 你接受还是拒绝？
- 最后机会 → 你抓住还是放弃？

选择（内容）：
- 接受背叛 → 原谅/报复/沉默
- 接受诱惑 → 妥协/谈判/拒绝
- 抓住机会 → 冒险/稳妥/放弃

【套路 + 选择 = 完整情节】
```

## 六、总结

```
【已使用】5个经典套路
【需要加强】3个套路
【需要新增】4个套路

【目标】
每个章节都有套路张力
每个角色都有自己的套路弧线
每个选择都有套路支撑

【原则】
套路是工具，服务于故事
不要为了套路而套路
让套路自然融入情节
```

---

> **一句话总结：好的套路让玩家"猜到了开头，猜不到结尾"——他们知道背叛会发生，但不知道背叛的细节；他们知道诱惑存在，但不知道代价是什么。**
