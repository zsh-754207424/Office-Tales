# 02-三幕结构

## 三幕概览

```
┌─────────────────────────────────────────────────────────┐
│                    【第一幕】入职适应                    │
│                      第1-2章                             │
│                                                         │
│  目标：在公司站稳脚跟，获得初步认可                      │
│  冲突：同期竞争、领导观察、融入团队                      │
│  节拍：好奇 → 尝试 → 遭遇困难 → 第一个选择              │
│  🚪 第一扇门：抢功事件（第2章结尾）                      │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│                    【第二幕】成长与抉择                   │
│                      第3-4章                             │
│                                                         │
│  目标：证明自己价值 / 站队选择 / 度过危机                │
│  冲突：晋升竞争、派系斗争、裁员危机                      │
│  节拍：希望 → 努力 → 获得盟友 → 遭遇背叛 → 危机         │
│  🚪 第二扇门：裁员名单公布（第4章结尾）                  │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│                    【第三幕】结局收束                     │

│                                                         │
│  目标：最终命运的揭晓                                    │
│  冲突：与领导对话 / 回顾选择 / 价值观呈现                │
│  节拍：紧张 → 揭示 → 选择 → 代价展示 → 反思              │
│  💥 撞击式结局：揭示成功/代价/意义                       │
└─────────────────────────────────────────────────────────┘
```

## 第一幕详细设计

### 时间跨度：入职第1天 - 第2周

### 核心任务
```
1. 认识环境（公司、部门、同事）
2. 建立初步人际关系
3. 获得领导的第一印象
4. 了解基本规则和潜规则
```

### 第一幕节拍

```
【节拍1：入职第一天】好奇与期待
├── 入职手续（小刘HR）
├── 部门介绍（李总监）
├── 认识同事（张姐、王浩、小林）
└── 第一印象建立

【节拍2：初步接触】尝试融入
├── 午餐选择（关键）
├── 工作安排
└── 同事互动

【节拍3：第一个任务】遭遇挑战
├── 李总监布置任务
├── 遇到问题
├── 选择应对方式
└── 任务完成度

【节拍4：抢功事件】关系转折（第一扇门）
├── 王浩的行为
├── 你的反应
├── 关系变化
└── 第一幕结束
```

### 第一幕的选择节点

| 选择 | 选项 | 影响 |
|------|------|------|
| 午餐 | 接受张姐/拒绝/自己吃 | 张姐好感±15 |
| 汇报 | 主动/被动/不汇报 | 李总监信任±20 |
| 态度 | 热情/冷淡/中性 | 整体印象 |
| 抢功应对 | 质问/沉默/私下/收集证据 | 王浩敌意+10~50 |

### 第一幕的情感曲线

```
        😊
       /  \
      /    \      😰
     /      \    /  \
    /   🤔   \  /    \
   /          \/      \
  ───────────────────────→ 时间
  入职    适应    任务    抢功
  好奇    尝试    紧张    愤怒
```

## 第二幕详细设计

### 时间跨度：第3周 - 第3个月

### 核心任务
```
1. 证明工作能力
2. 建立核心人际关系
3. 应对派系斗争
4. 度过裁员危机
```

### 第二幕节拍

```
【第一阶段：成长（第3章）】

节拍1：季度评估
├── 季度总结大会
├── 秦总远观（翻版出现）
├── 工作成果展示
└── 功劳分配选择

节拍2：关系深化
├── 张姐的考验
├── 王浩的持续对抗
├── 陈叔的情报（解锁）
└── Linda的出现

节拍3：派系暗流
├── 了解派系格局
├── 站队选择
├── 当众表态
└── 盟友关系确立

【第二阶段：危机（第4章）】

节拍4：裁员预警
├── 陈叔的核心情报
├── 小刘HR的暗示
├── 公司氛围变化
└── 王副总的出手

节拍5：最终对抗
├── 王副总拉拢/威胁
├── 裁员站队
├── 与王浩的最后博弈
└── 名单公布（第二扇门）
```

### 第二幕的选择节点

| 选择 | 选项 | 影响 |
|------|------|------|
| 功劳分配 | 强调团队/展示个人/分享给王浩 | 张姐±15/李总监±15/王浩-10 |
| 当众发言 | 积极/沉默/提建议 | 全员±15/高风险高回报 |
| 站队 | 支持张姐/中立/疏远 | 张姐±25/无变化/-20 |
| 王副总拉拢 | 接受/拒绝/不表态 | 李总监信任归零/敌意/两边不信任 |
| 对抗王浩 | 正面竞争/揭发/和解 | 凭数值/双输/罕见 |

## 第三幕详细设计

### 时间跨度：第4个月 - 第5个月（结局）

### 核心任务
```
1. 揭晓最终命运
2. 与关键人物对话
3. 揭示选择的意义
4. 展示"成功"的代价
```

### 第三幕节拍

```
节拍1：命运揭晓
├── 裁员结果公布
├── 你的名字在/不在名单上
└── 等待最终对话

节拍2：领导对话
├── 与李总监/王副总对话
├── 回顾你的表现
└── 给出评价

节拍3：终极对话（如晋升）
├── 秦总的评价
├── 关于"成功"的对话
└── 你想成为什么样的人？

节拍4：结局对照
├── 小林的结局
├── 其他角色的命运
└── 你的选择带来的后果

节拍5：撞击式结局
├── 揭示代价
├── 展示你失去/得到的东西
├── 玩家的最终选择
└── 反思：这一切值得吗？
```

### 第三幕的选择节点

| 选择 | 场景 | 影响 |
|------|------|------|
| 价值观选择 | 秦总问你"你想成为什么样的人？" | 决定结局的最终走向 |

## 两扇门的设计

### 第一扇门：抢功事件（第2章结尾）

```
【门的功能】
- 不可逆的关系转折
- 故事从此只能向前
- 决定与王浩的关系基调

【门的内容】
王浩把你的方案给领导看
领导表扬了他，你被忽视
无论你怎么选，和王浩的关系都会改变

【门的条件】
触发条件：完成第2章任务后自动触发
无法跳过：是

【门的选择】
A. 当面质问 → 公开对立，王浩敌意+30
B. 沉默接受 → 被动积累，王浩敌意+10
C. 私下找领导 → 私下博弈，王浩敌意+50
D. 收集证据 → 隐藏路线，保留筹码
```

### 第二扇门：裁员名单（第4章结尾）

```
【门的功能】
- 命运的最终转折
- 故事进入最终阶段
- 决定结局的大方向

【门的内容】
裁员名单公布
你看到自己的名字在/不在上面
无论结果如何，故事进入揭示阶段

【门的条件】
触发条件：完成第4章所有选择后自动触发
无法跳过：是

【门的可能】
名单上有你 → C类结局（离职）
名单上有王浩 → A类结局（晋升）
名单上有你和王浩 → B类结局（稳定）
```

## 撞击式结局设计

### 什么是撞击式结局

```
【传统结局】
结局 → 故事结束 → 反思

【撞击式结局】
结局 → 代价揭示 → 玩家选择 → 反思
          ↓
     "你想成为什么样的人？"
```

### 撞击式结局的三个层次

```
【第一撞击：命运】
你成功了/失败了/离开了
→ 揭示你的选择带来的结果

【第二撞击：代价】
你得到了/失去了
→ 揭示成功/失败背后的代价

【第三撞击：选择】
你想成为什么样的人？
→ 让玩家反思，并做出最终选择
```

---

## 一句话总结

> **三幕结构的灵魂：每一幕都有明确目标、冲突和转折，让玩家体验完整的情感旅程**
